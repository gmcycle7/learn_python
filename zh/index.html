學習 Python — 完整初學者指南

一個以實作範例驅動、從零開始學習 Python 的教學網站

歡迎！本網站是一門結構化、自主學習的課程，專為想要以正確方式學習 Python 的完全初學者設計 — 透過閱讀清晰的說明和研究實際的範例來學習。無論您從未寫過任何一行程式碼，還是您有 MATLAB 背景並希望快速掌握 Python，每個章節都是為您量身打造的。 本課程不假設您具備任何程式設計的先備知識；每個概念都從頭開始介紹。

每一頁都包含了附有詳細註解的程式碼片段，您可以複製並自行執行。 概念按照邏輯順序層層遞進，但您也可以隨時透過側邊欄導覽跳到任何 您感興趣的主題。

章節 A — 什麼是 Python

章節 A

什麼是 Python？

了解為什麼 Python 是世界上最受歡迎的程式語言之一。 認識它的歷史、設計哲學，以及它被廣泛應用的領域 — 從資料科學和機器學習到網頁開發和自動化。

章節 B — 入門設定

章節 B

入門設定

安裝 Python、設定您的開發環境，並執行您的第一個腳本。 涵蓋編輯器的選擇（VS Code、PyCharm、Jupyter）、終端機的使用， 以及了解 Python 檔案的結構。

章節 C — Python 基礎

課程的核心部分。這九頁內容將帶您逐一學習 Python 的每一個基本概念。

章節 C1

輸出、註解、變數、數字

從 print() 開始，學習註解的用法、建立變數， 並探索 Python 的數值型別 — 整數、浮點數和基本算術運算子。

章節 C2

字串、布林值、型別轉換

使用字串處理文字、了解布林邏輯（True / False），並學習如何使用 int()、str() 和 float() 進行型別轉換。

章節 C3

串列、元組、集合

使用串列儲存資料集合、了解不可變的元組，並使用集合 進行快速的成員檢測。學習索引、切片和常見的集合方法。

章節 C4

字典

掌握 Python 的鍵值資料結構。建立、存取、更新和遍歷 字典 — Python 中最強大且最常使用的型別之一。

章節 C5

if / elif / else

使用條件陳述式控制程式流程。學習比較運算子、 邏輯運算子（and、or、not）， 以及如何撰寫簡潔、易讀的分支邏輯。

章節 C6

for、while、break、continue、pass

使用 for 和 while 迴圈重複執行動作。學習如何 遍歷串列、字串和範圍，並使用 break、continue 和 pass 控制迴圈執行。

章節 C7

函式、回傳值、作用域、Lambda

使用函式撰寫可重複使用的程式碼。了解參數、回傳值、變數 作用域、預設引數、*args / **kwargs， 以及單行的 lambda 運算式。

章節 C8

更多內建功能與技巧

進階學習 input()、錯誤處理基礎、f-string、切片、 串列推導式、enumerate()、zip()、 range()、排序，以及 map() / filter()。

章節 C9

檔案讀寫、try/except、匯入模組

讀取和寫入檔案、使用 try / except 優雅地處理錯誤，並學習如何匯入和使用 Python 豐富的 標準函式庫和第三方套件。

章節 D — 常用模式

章節 D1

常用模式

探索在實際 Python 程式中反覆出現的程式碼模式 — 累加器迴圈、字典計數、巢狀迭代、防護子句 等等。認識這些模式能加速您的學習。

章節 D2

百大常用模式

精選的 100 個簡短 Python 模式參考，您將一再使用它們。 每一個都是獨立的範例，您可以複製、改編並組合 到您自己的專案中。

章節 E — MATLAB 轉 Python

章節 E1

MATLAB 轉 Python

為 MATLAB 使用者轉換到 Python 而設計的專門指南。涵蓋 理念上的差異、等效的工作流程，以及取代 MATLAB 內建工具箱的 函式庫（NumPy、SciPy、Matplotlib）。

章節 E2

快速對照指南

數十種常見操作的 MATLAB 與 Python 程式碼並列對照 — 陣列、繪圖、線性代數、檔案讀寫等等。在您撰寫程式碼時 可以將此頁面作為快速參考。

章節 F — 語言比較

章節 F

語言比較

了解 Python 與其他熱門語言（如 JavaScript、C++、Java 和 R）的比較。 理解 Python 在速度、語法簡潔性和生態系統 方面的優劣取捨。

章節 G — 迷你專案

章節 G

迷你專案

透過引導式迷你專案實踐您的技能：猜數字遊戲、 待辦事項應用程式、檔案整理器、網頁爬蟲等等。每個專案 都綜合運用了基礎章節中的多個概念。

章節 H — 練習

章節 H

練習題

透過數十道按難度分類的練習題磨練您的技能。 每道題目都附有提示和完整解答，讓您能檢查自己的作業 並從不同的解題方法中學習。

章節 I — 最佳實踐

章節 I1

最佳實踐

撰寫乾淨、易讀且易於維護的 Python 程式碼。涵蓋 PEP 8 風格、 有意義的命名、DRY 原則、虛擬環境、專案結構， 以及區分初學者與專業人士的良好習慣。

章節 I2

常見錯誤

您最常遇到的錯誤訊息實用指南 — SyntaxError、IndentationError、 TypeError、KeyError 等等 — 附有 每種錯誤的解說和修正方法。

章節 J — 附錄

章節 J

速查表

可列印的快速參考，涵蓋 Python 的語法、資料型別、 運算子、字串方法、串列方法、字典方法、常用模組， 以及常用的單行程式碼。

附錄

附錄

Python 自動化應用

了解 Python 如何幫您省下數小時的重複工作。學習自動化 檔案管理、試算表處理、電子郵件發送、網頁爬蟲， 以及排程任務，附有可立即使用的實用腳本。

從哪裡開始

如果您是程式設計的新手，建議您按照章節順序學習。 以下是建議的閱讀路徑：

建議閱讀順序
  1. 什麼是 Python？ — 了解您要學習的內容以及為什麼它很重要。
  2. 入門設定 — 安裝 Python 並設定編輯器，以便跟著操作。
  3. 輸出、變數、數字 — 撰寫並執行您的第一段程式碼。
  4. 字串、布林值、型別 — 處理文字和 True/False 值。
  5. 串列、元組、集合 — 儲存和組織資料。
  6. 字典 — 將鍵對應到值。
  7. 條件判斷 — 在程式碼中做出決策。
  8. 迴圈 — 高效地重複執行動作。
  9. 函式 — 撰寫可重複使用的邏輯區塊。
  10. 更多內建功能 — 用必備的技巧完善您的工具包。
  11. 檔案與模組 — 讀取檔案和使用函式庫。
  12. 常用模式 — 識別並應用常見的慣用寫法。
  13. 練習 — 透過實作練習鞏固所學。
  14. 迷你專案 — 建造一些真正的東西。
來自 MATLAB？

如果您已經熟悉 MATLAB，請先閱讀 MATLAB 轉 Python 以獲得 概念性的總覽，然後在學習基礎時使用 快速對照指南 作為並列參考。您會發現許多 MATLAB 概念可以直接對應到 Python — 只是語法不同而已。

提示：做中學

不要只是閱讀範例 — 親手輸入它們、執行它們、修改它們， 甚至故意弄壞它們。學習程式語言最快的方式就是 每天都寫程式碼，即使只有幾行也好。

用心為初學者打造。祝您寫程式愉快！
