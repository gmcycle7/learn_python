字典

字典是 Python 中最強大且最常用的資料結構之一。如果串列讓你儲存有序的值序列， 那麼字典則讓你儲存鍵值對 — 將標籤（鍵）與資料（值）連結起來。 一旦你理解了字典，你會發現它們無處不在： 設定檔、JSON 資料、函式的關鍵字引數等等。

MATLAB 比較： MATLAB 有 containers.Map，其運作方式類似 — 你透過鍵而非數字位置來儲存和索引值。然而， containers.Map 在 MATLAB 程式碼中相對少見。 在 Python 中，字典（dict）是語言的核心。 它們被廣泛使用 — 用於 JSON 資料、 函式 **kwargs、底層的類別屬性、 模組命名空間等等。掌握字典是必不可少的。

1. 什麼是字典？

想像一本現實世界的字典：你查找一個單字 （鍵）然後找到它的定義（值）。Python 的字典運作方式完全相同 — 每個條目都是一個 鍵值對，你使用鍵來檢索對應的值。

使用 {} 建立字典

建立字典最常見的方式是使用大括號， 用冒號分隔鍵和值： 

# 一個簡單的字典：英文單字 -> 西班牙文翻譯
english_to_spanish = {
    "hello": "hola",
    "goodbye": "adiós",
    "please": "por favor",
    "thank you": "gracias"
}

print(english_to_spanish)
# {'hello': 'hola', 'goodbye': 'adiós', 'please': 'por favor', 'thank you': 'gracias'}

使用 dict() 建立字典

你也可以使用內建的 dict() 建構函式來建立字典。當你的鍵是看起來像有效 Python 變數名稱的簡單字串時，這種方式特別好用： 

# 使用 dict() 搭配關鍵字引數
person = dict(name="Alice", age=30, city="London")
print(person)
# {'name': 'Alice', 'age': 30, 'city': 'London'}

# 使用 dict() 搭配元組串列
pairs = [("x", 10), ("y", 20), ("z", 30)]
coordinates = dict(pairs)
print(coordinates)
# {'x': 10, 'y': 20, 'z': 30}

空字典

# 建立空字典的兩種方式
empty1 = {}
empty2 = dict()

print(type(empty1))  # <class 'dict'>
print(len(empty1))   # 0
注意： {} 建立的是空 字典，不是空集合。要建立空集合， 你必須使用 set()。這是初學者常見的混淆來源。

鍵的規則：鍵必須是不可變的

字典的鍵必須是不可變的（不可更改的） 型別。這表示你可以使用：

  • 字串 — 最常見的鍵型別
  • 數字（int、float） — 完全有效
  • 元組 — 僅當它們包含不可變元素時
  • 布林值 — 技術上有效但很少有用

你不能使用串列、字典或集合作為鍵， 因為它們是可變的：

# 有效的鍵
valid = {
    "name": "Alice",       # 字串鍵
    42: "the answer",      # 整數鍵
    3.14: "pi",            # 浮點數鍵
    (1, 2): "coordinates"  # 元組鍵
}

# 無效的鍵 - 這會引發 TypeError
# bad = {[1, 2]: "a list as key"}  # TypeError: unhashable type: 'list'
為什麼鍵必須是不可變的？ Python 使用一種稱為 雜湊的技術來快速查找鍵。雜湊是從鍵的值計算出的 固定大小數字。如果鍵在被加入後可以改變（像是一個被附加了項目的串列）， 它的雜湊值也會改變，Python 就無法再找到它。不可變型別保證了雜湊值保持不變。

另一方面，值可以是任何東西 — 字串、數字、串列、其他字典、函式、物件等。

2. 存取值

從字典中取得值有兩種主要方式：方括號表示法和 .get() 方法。選擇哪一種取決於你是否預期鍵一定存在。

方括號表示法：dict[key]

使用方括號加上鍵來檢索值。如果鍵不存在， Python 會引發 KeyError： 

student = {"name": "Bob", "grade": "A", "age": 21}

print(student["name"])   # Bob
print(student["grade"])  # A

# 嘗試存取不存在的鍵：
# print(student["email"])  # KeyError: 'email'

.get() 方法：安全存取

.get() 方法在鍵不存在時回傳 None， 而不是讓你的程式崩潰： 

student = {"name": "Bob", "grade": "A", "age": 21}

# .get() 在鍵不存在時回傳 None
print(student.get("email"))       # None
print(student.get("name"))        # Bob

# 你可以提供預設值作為第二個引數
print(student.get("email", "N/A"))    # N/A
print(student.get("grade", "N/A"))    # A（鍵存在，所以忽略預設值）
何時使用哪一種：
  • 當你確定鍵存在時，使用 dict[key]， 或者當鍵缺失表示一個應該大聲報錯的 bug 時。
  • 當鍵可能不存在，而你想優雅地處理時， 使用 dict.get(key)。
  • 當你想要為缺失的鍵提供一個備用值時， 使用 dict.get(key, default)。

3. 修改字典

字典是可變的 — 你可以在建立後 新增、更改和移除鍵值對。

新增和更新值

student = {"name": "Bob", "age": 21}

# 新增一個鍵值對
student["email"] = "bob@example.com"
print(student)
# {'name': 'Bob', 'age': 21, 'email': 'bob@example.com'}

# 更新現有的值
student["age"] = 22
print(student)
# {'name': 'Bob', 'age': 22, 'email': 'bob@example.com'}
注意： 新增和更新的語法是相同的 — dict[key] = value。如果鍵已經存在，舊值會被替換。 如果不存在，則建立新條目。

.update() 方法

使用 .update() 將另一個字典（或可迭代的鍵值對） 合併到現有字典中。已存在的鍵會被覆寫；新鍵會被新增： 

student = {"name": "Bob", "age": 21}

# 用另一個字典更新
student.update({"age": 22, "grade": "A", "email": "bob@example.com"})
print(student)
# {'name': 'Bob', 'age': 22, 'grade': 'A', 'email': 'bob@example.com'}

# 用關鍵字引數更新
student.update(city="New York", age=23)
print(student)
# {'name': 'Bob', 'age': 23, 'grade': 'A', 'email': 'bob@example.com', 'city': 'New York'}

移除條目

Python 提供了多種從字典中移除鍵值對的方式： 

student = {"name": "Bob", "age": 22, "grade": "A", "email": "bob@example.com"}

# del — 移除指定的鍵（如果不存在則引發 KeyError）
del student["email"]
print(student)
# {'name': 'Bob', 'age': 22, 'grade': 'A'}

# .pop(key) — 移除並回傳值（如果不存在則引發 KeyError）
grade = student.pop("grade")
print(grade)     # A
print(student)   # {'name': 'Bob', 'age': 22}

# .pop(key, default) — 回傳預設值而非引發 KeyError
missing = student.pop("phone", "not found")
print(missing)   # not found

# .popitem() — 移除並回傳最後插入的鍵值對
student["city"] = "London"
student["year"] = 3
last = student.popitem()
print(last)      # ('year', 3)
print(student)   # {'name': 'Bob', 'age': 22, 'city': 'London'}

# .clear() — 移除所有條目
student.clear()
print(student)   # {}
常見錯誤： 在沒有先檢查鍵是否存在的情況下使用 del dict[key]。如果鍵不存在， 你會得到 KeyError。先用 if key in dict 檢查，或使用 dict.pop(key, None) 來安全移除。

4. 字典方法

.keys()、.values() 和 .items()

這三個方法讓你以不同方式檢視字典的內容： 

fruit_colors = {"apple": "red", "banana": "yellow", "grape": "purple"}

# .keys() — 所有鍵
print(fruit_colors.keys())
# dict_keys(['apple', 'banana', 'grape'])

# .values() — 所有值
print(fruit_colors.values())
# dict_values(['red', 'yellow', 'purple'])

# .items() — 所有鍵值對（以元組形式）
print(fruit_colors.items())
# dict_items([('apple', 'red'), ('banana', 'yellow'), ('grape', 'purple')])

# 需要時轉換為串列
keys_list = list(fruit_colors.keys())
print(keys_list)  # ['apple', 'banana', 'grape']
這些是視圖，不是副本。 .keys()、.values() 和 .items() 回傳的物件是視圖， 它們會反映字典的變更。如果你向字典新增一個鍵， 視圖會自動包含它。

.copy() — 淺層複製

original = {"a": 1, "b": 2, "c": 3}
duplicate = original.copy()

duplicate["d"] = 4
print(original)   # {'a': 1, 'b': 2, 'c': 3}（未改變）
print(duplicate)  # {'a': 1, 'b': 2, 'c': 3, 'd': 4}
淺層複製警告： .copy() 建立的是 淺層複製。如果你的字典包含可變值 （如串列或其他字典），那些內部物件是 共享的，不是被複製的。如果你需要一個完全獨立的副本， 使用 import copy; copy.deepcopy(dict)。

使用 in 檢查鍵是否存在

student = {"name": "Bob", "age": 22}

print("name" in student)     # True
print("email" in student)    # False
print("Bob" in student)      # False — "in" 檢查的是鍵，不是值

# 常見模式：存取前先檢查
if "email" in student:
    print(student["email"])
else:
    print("No email on file")
MATLAB 比較： 在 MATLAB 中使用 containers.Map 時，你用 isKey(map, key) 來檢查鍵是否存在。在 Python 中，in 關鍵字 讀起來更自然：if key in dict。

len() — 鍵值對數量

student = {"name": "Bob", "age": 22, "grade": "A"}
print(len(student))  # 3

5. 遍歷字典

你經常需要迭代字典。Python 提供了幾種簡潔的方式來做到這一點。

遍歷鍵（預設）

fruit_colors = {"apple": "red", "banana": "yellow", "grape": "purple"}

# 預設情況下，迭代字典會給你鍵
for fruit in fruit_colors:
    print(fruit)

# 輸出：
# apple
# banana
# grape

同時遍歷鍵和值

fruit_colors = {"apple": "red", "banana": "yellow", "grape": "purple"}

# .items() 同時給你鍵和值
for fruit, color in fruit_colors.items():
    print(f"The {fruit} is {color}")

# 輸出：
# The apple is red
# The banana is yellow
# The grape is purple
為什麼 .items() 是最常見的模式： 大多數時候當你遍歷字典時，你需要同時使用鍵和值。使用 for key, value in dict.items() 比先遍歷鍵再用 dict[key] 取得每個值更簡潔且更有效率。

僅遍歷值

scores = {"Alice": 95, "Bob": 87, "Charlie": 92}

# 如果你只關心值
for score in scores.values():
    print(score)

# 輸出：
# 95
# 87
# 92

# 適合用於計算
total = sum(scores.values())
average = total / len(scores)
print(f"Average score: {average:.1f}")  # Average score: 91.3

6. 巢狀字典

字典的值可以是任何東西 — 包括其他字典。 這稱為巢狀，在真實世界的資料中極為常見（想想 JSON）。

字典中的字典

# 學生記錄 — 每個學生是一個巢狀字典
students = {
    "alice": {
        "full_name": "Alice Johnson",
        "age": 22,
        "grades": {"math": 95, "physics": 88, "cs": 92}
    },
    "bob": {
        "full_name": "Bob Smith",
        "age": 21,
        "grades": {"math": 78, "physics": 85, "cs": 90}
    }
}

存取巢狀值

# 串連方括號以深入巢狀字典
print(students["alice"]["full_name"])           # Alice Johnson
print(students["bob"]["grades"]["physics"])     # 85

# 對於深層巢狀資料，使用 .get() 更安全
math_grade = students.get("charlie", {}).get("grades", {}).get("math", "N/A")
print(math_grade)  # N/A（charlie 不存在）

真實世界範例：設定檔

# 應用程式設定（類似於 JSON 設定檔中的內容）
config = {
    "database": {
        "host": "localhost",
        "port": 5432,
        "name": "myapp_db",
        "credentials": {
            "user": "admin",
            "password": "secret123"
        }
    },
    "logging": {
        "level": "INFO",
        "file": "/var/log/myapp.log",
        "max_size_mb": 50
    },
    "features": {
        "dark_mode": True,
        "notifications": False
    }
}

# 存取深層巢狀的值
db_host = config["database"]["host"]
log_level = config["logging"]["level"]
print(f"Connecting to {db_host}, log level: {log_level}")
# Connecting to localhost, log level: INFO
JSON 與字典： JSON（JavaScript 物件表示法） 是 Web API 最常見的資料格式。當你在 Python 中使用 import json 載入 JSON 資料時，它會變成一個字典 （或字典的串列）。這是字典如此重要的原因之一。

7. 字典推導式

就像串列推導式一樣，Python 支援 字典推導式 — 一種從可迭代物件簡潔地建立字典的方式。

基本語法

# 語法：{鍵運算式: 值運算式 for 項目 in 可迭代物件}

# 建立一個平方數字典
squares = {n: n ** 2 for n in range(1, 6)}
print(squares)
# {1: 1, 2: 4, 3: 9, 4: 16, 5: 25}

實用範例

# 將單字串列轉換為單字長度字典
words = ["hello", "world", "python", "code"]
word_lengths = {word: len(word) for word in words}
print(word_lengths)
# {'hello': 5, 'world': 5, 'python': 6, 'code': 4}

# 篩選：只包含符合條件的項目
scores = {"Alice": 95, "Bob": 67, "Charlie": 82, "Diana": 91}
high_scores = {name: score for name, score in scores.items() if score >= 85}
print(high_scores)
# {'Alice': 95, 'Diana': 91}

# 交換鍵和值
original = {"a": 1, "b": 2, "c": 3}
swapped = {v: k for k, v in original.items()}
print(swapped)
# {1: 'a', 2: 'b', 3: 'c'}

# 轉換值：攝氏轉華氏
celsius = {"London": 15, "Paris": 18, "Tokyo": 22, "Sydney": 25}
fahrenheit = {city: round(temp * 9/5 + 32, 1) for city, temp in celsius.items()}
print(fahrenheit)
# {'London': 59.0, 'Paris': 64.4, 'Tokyo': 71.6, 'Sydney': 77.0}
MATLAB 比較： MATLAB 沒有字典推導式。 你通常需要使用迴圈配合 containers.Map 或逐一建構結構體欄位。 Python 的推導式讓你能用單一可讀的表達式建立整個字典。

8. 常見用途

以下是在真實 Python 程式碼中反覆出現的四種模式。 掌握這些會讓你成為更有效率的程式設計師。

計算出現次數

字典最常見的用途之一是計算序列中每個項目出現的次數： 

# 計算字串中的字元頻率
text = "mississippi"
freq = {}
for char in text:
    freq[char] = freq.get(char, 0) + 1

print(freq)
# {'m': 1, 'i': 4, 's': 4, 'p': 2}

# 或使用內建的 collections.Counter（捷徑！）
from collections import Counter
freq2 = Counter(text)
print(freq2)
# Counter({'i': 4, 's': 4, 'p': 2, 'm': 1})
為什麼 freq.get(char, 0) + 1 有效： 如果字元還不在字典中， .get(char, 0) 回傳 0。我們加上 1，所以任何字元的首次出現從 1 開始。 在後續遇到時，會檢索當前計數並遞增。 這是一種簡潔的替代方案，不需要先檢查 if char in freq。

資料分組

# 按成績分組學生
students = [
    ("Alice", "A"), ("Bob", "B"), ("Charlie", "A"),
    ("Diana", "B"), ("Eve", "A"), ("Frank", "C")
]

groups = {}
for name, grade in students:
    if grade not in groups:
        groups[grade] = []
    groups[grade].append(name)

print(groups)
# {'A': ['Alice', 'Charlie', 'Eve'], 'B': ['Bob', 'Diana'], 'C': ['Frank']}

# 使用 .setdefault() 更簡潔的版本
groups2 = {}
for name, grade in students:
    groups2.setdefault(grade, []).append(name)

print(groups2)
# {'A': ['Alice', 'Charlie', 'Eve'], 'B': ['Bob', 'Diana'], 'C': ['Frank']}

查找值（取代冗長的 if/elif 鏈）

# 取代這樣寫：
def get_day_name_ugly(day_num):
    if day_num == 1:
        return "Monday"
    elif day_num == 2:
        return "Tuesday"
    elif day_num == 3:
        return "Wednesday"
    # ... 等等

# 使用字典查找：
day_names = {
    1: "Monday", 2: "Tuesday", 3: "Wednesday",
    4: "Thursday", 5: "Friday", 6: "Saturday", 7: "Sunday"
}

def get_day_name(day_num):
    return day_names.get(day_num, "Invalid day")

print(get_day_name(3))  # Wednesday
print(get_day_name(9))  # Invalid day
效能提示： 字典查找平均是 O(1) — 無論字典有多少鍵，時間都是固定的。一長串的 if/elif 語句是 O(n) — 隨著你增加更多情況會變慢。對於大型查找表，字典既更簡潔又更快。

設定儲存

# 常見模式：使用字典來儲存設定
default_settings = {
    "theme": "light",
    "font_size": 14,
    "auto_save": True,
    "language": "en",
    "max_retries": 3
}

# 使用者覆寫部分設定
user_settings = {
    "theme": "dark",
    "font_size": 16
}

# 合併：使用者設定覆寫預設值
final_settings = {**default_settings, **user_settings}
print(final_settings)
# {'theme': 'dark', 'font_size': 16, 'auto_save': True, 'language': 'en', 'max_retries': 3}
** 解包運算子： {**dict1, **dict2} 語法透過解包兩個字典來建立新字典。 如果它們有相同的鍵，第二個字典的值會勝出。 這是一種簡潔、Pythonic 的字典合併方式。在 Python 3.9+ 中， 你也可以使用 | 運算子：dict1 | dict2。

常見錯誤

錯誤 會發生什麼 修正方法
dict[missing_key] KeyError 使用 dict.get(key) 或用 in 檢查
使用串列作為鍵 TypeError: unhashable type 先轉換為元組：tuple(my_list)
用 {} 建立空集合 建立的是空字典 使用 set() 建立空集合
在遍歷字典時修改它 RuntimeError: dictionary changed size during iteration 遍歷副本：for k in list(dict.keys()):
假設字典在舊版 Python 中是有序的 插入順序僅在 Python 3.7+ 中保證 使用 Python 3.7+（或 collections.OrderedDict）
檢查 value in dict 檢查的是鍵，不是值 使用 value in dict.values() 來檢查值

快速參考

操作 語法 備註
建立 d = {"a": 1, "b": 2} 最常見的方式
存取 d["a"] 鍵不存在時引發 KeyError
安全存取 d.get("a", default) 鍵不存在時回傳預設值
新增 / 更新 d["c"] = 3 新增或覆寫
合併 d.update(other) 就地修改 d
刪除 del d["a"] 鍵不存在時引發 KeyError
安全刪除 d.pop("a", None) 鍵不存在時回傳 None
檢查鍵 "a" in d 回傳 True/False
所有鍵 d.keys() 回傳視圖物件
所有值 d.values() 回傳視圖物件
所有鍵值對 d.items() 回傳 (key, value) 元組
長度 len(d) 鍵值對數量
複製 d.copy() 淺層複製
清除 d.clear() 移除所有條目
推導式 {k: v for k, v in ...} 從可迭代物件建立

練習 1：建立電話簿

建立一個名為 phone_book 的字典，包含至少 三個姓名到電話號碼的條目。然後列印特定人員的電話號碼、 新增一個條目，並刪除一個條目。

顯示解答 
phone_book = {
    "Alice": "555-1234",
    "Bob": "555-5678",
    "Charlie": "555-9012"
}

# 列印 Bob 的號碼
print(phone_book["Bob"])  # 555-5678

# 新增一個條目
phone_book["Diana"] = "555-3456"

# 刪除 Charlie
del phone_book["Charlie"]

print(phone_book)
# {'Alice': '555-1234', 'Bob': '555-5678', 'Diana': '555-3456'}

練習 2：單字頻率計數器

給定字串 "the cat sat on the mat the cat"， 寫程式碼計算每個單字出現的次數。 將結果存在字典中。

顯示解答 
sentence = "the cat sat on the mat the cat"
words = sentence.split()

word_count = {}
for word in words:
    word_count[word] = word_count.get(word, 0) + 1

print(word_count)
# {'the': 3, 'cat': 2, 'sat': 1, 'on': 1, 'mat': 1}

練習 3：反轉字典

寫程式碼將字典 {"a": 1, "b": 2, "c": 3} 的鍵和值交換，建立新字典： {1: "a", 2: "b", 3: "c"}。 使用字典推導式。

顯示解答 
original = {"a": 1, "b": 2, "c": 3}
inverted = {v: k for k, v in original.items()}
print(inverted)
# {1: 'a', 2: 'b', 3: 'c'}

練習 4：合併兩個字典

你有兩個字典： 

defaults = {"color": "blue", "size": 10, "visible": True}
overrides = {"color": "red", "border": 2}

建立一個包含兩者所有鍵的新字典，其中 overrides 對於共享鍵的值優先於 defaults。

顯示解答 
defaults = {"color": "blue", "size": 10, "visible": True}
overrides = {"color": "red", "border": 2}

# 方法 1：** 解包
merged = {**defaults, **overrides}
print(merged)
# {'color': 'red', 'size': 10, 'visible': True, 'border': 2}

# 方法 2：.update()（就地修改 defaults）
defaults_copy = defaults.copy()
defaults_copy.update(overrides)
print(defaults_copy)
# {'color': 'red', 'size': 10, 'visible': True, 'border': 2}

# 方法 3：| 運算子（Python 3.9+）
# merged = defaults | overrides

練習 5：學生成績平均

給定以下學生成績的巢狀字典，計算並列印 每位學生的平均成績： 

students = {
    "Alice": {"math": 90, "science": 85, "english": 92},
    "Bob": {"math": 78, "science": 82, "english": 88},
    "Charlie": {"math": 95, "science": 91, "english": 87}
}
顯示解答 
students = {
    "Alice": {"math": 90, "science": 85, "english": 92},
    "Bob": {"math": 78, "science": 82, "english": 88},
    "Charlie": {"math": 95, "science": 91, "english": 87}
}

for name, grades in students.items():
    avg = sum(grades.values()) / len(grades)
    print(f"{name}: average = {avg:.1f}")

# 輸出：
# Alice: average = 89.0
# Bob: average = 82.7
# Charlie: average = 91.0

練習 6：按首字母分組

給定一個單字串列，按首字母將它們分組到字典中。 例如，["apple", "banana", "avocado", "blueberry", "cherry", "apricot"] 應該產生： 

{"a": ["apple", "avocado", "apricot"], "b": ["banana", "blueberry"], "c": ["cherry"]}
顯示解答 
words = ["apple", "banana", "avocado", "blueberry", "cherry", "apricot"]

groups = {}
for word in words:
    first_letter = word[0]
    groups.setdefault(first_letter, []).append(word)

print(groups)
# {'a': ['apple', 'avocado', 'apricot'], 'b': ['banana', 'blueberry'], 'c': ['cherry']}

# 使用一般 if 檢查的替代方案：
groups2 = {}
for word in words:
    letter = word[0]
    if letter not in groups2:
        groups2[letter] = []
    groups2[letter].append(word)

總結

  • 字典儲存鍵值對。鍵必須是不可變的；值可以是任何東西。
  • 當你確定鍵存在時使用 dict[key]； 使用 dict.get(key, default) 進行安全存取。
  • .keys()、.values() 和 .items() 讓你檢視字典的內容。
  • 最常見的迴圈模式是 for key, value in dict.items()。
  • 字典可以巢狀 — 值可以是其他字典、串列等。
  • 字典推導式讓你從可迭代物件簡潔地建立字典。
  • 常見模式包括計數、分組、查找和設定儲存。
  • 字典是 Python 的核心 — 它們驅動 JSON 處理、 函式關鍵字引數以及語言的大部分內部機制。
為什麼這很重要： 字典可以說是 Python 中 僅次於串列的最重要資料結構。無論你是在解析 API 回應、 建立設定物件、計算項目還是組織資料，你每天都會使用它們。 你在這裡學到的模式 — 特別是使用 .get() 計數、 使用 .setdefault() 分組和推導式 — 幾乎會出現在你參與的每個 Python 專案中。
