檔案與模組

本頁涵蓋兩項基本技能：處理磁碟上的檔案 （讀取和寫入資料），以及使用模組和匯入來組織你的程式碼。 我們也會介紹 try / except 錯誤處理 — 因為處理檔案是最常出問題的場景之一。 學完本頁後，你將能夠讀寫文字檔案、優雅地處理錯誤， 以及使用模組來組織你的 Python 專案。

MATLAB 使用者注意：MATLAB 使用 fopen、fread、fprintf 和 fclose 來進行檔案 I/O。Python 的方法在精神上 類似，但使用不同的語法 — 而且 with 陳述式消除了記住呼叫 fclose 的需要。 對於模組，MATLAB 依賴 addpath 並將函式放在 獨立的 .m 檔案中。Python 使用 import，這更加明確和強大。

1. 讀取檔案

open() 函式與檔案模式

Python 的內建 open() 函式是檔案操作的入口。 它回傳一個檔案物件，你可以用它來讀取或寫入檔案。 第二個引數是模式：

模式 說明 檔案必須存在？
"r" 讀取（預設）。開啟檔案進行讀取。 是
"w" 寫入。建立檔案或覆蓋現有內容。 否（會建立）
"a" 附加。在檔案末尾寫入，不會清除現有內容。 否（會建立）
"r+" 讀取和寫入。檔案必須已經存在。 是

使用 .read() 讀取整個檔案

# 將整個檔案讀入單一字串
with open("data.txt", "r") as f:
    content = f.read()

print(content)

.read() 方法將整個檔案作為一個字串回傳。 對於小檔案沒有問題，但對於非常大的檔案要小心 — 它們會消耗大量記憶體。

逐行讀取

Python 提供了幾種逐行讀取檔案的方式： 

# 方法 1：.readline() — 一次讀取一行
with open("data.txt", "r") as f:
    first_line = f.readline()   # 讀取第 1 行
    second_line = f.readline()  # 讀取第 2 行
    print(first_line.strip())   # .strip() 移除結尾的換行符
    print(second_line.strip())
# 方法 2：.readlines() — 回傳所有行的串列
with open("data.txt", "r") as f:
    lines = f.readlines()       # 例如 ["line 1\n", "line 2\n", "line 3\n"]

for line in lines:
    print(line.strip())
# 方法 3（最佳）：直接迭代檔案物件
with open("data.txt", "r") as f:
    for line in f:
        print(line.strip())
最佳實踐：方法 3（直接迭代）是最 Pythonic 且最 節省記憶體的方式。它一次讀取一行，不會將整個檔案載入記憶體， 使其對任何大小的檔案都是安全的。

with 陳述式（上下文管理器）

處理檔案時你應該始終使用 with 陳述式。 它保證當區塊結束時檔案會被正確關閉，即使發生錯誤也是如此： 

# 好 — with 陳述式（檔案自動關閉）
with open("data.txt", "r") as f:
    content = f.read()
# f 現在已關閉 — 不需要呼叫 f.close()

# 不好 — 手動開啟/關閉（容易忘記，容易出錯）
f = open("data.txt", "r")
content = f.read()
f.close()  # 如果在這行之前發生錯誤怎麼辦？
常見錯誤：忘記關閉檔案。如果你開啟了 很多檔案而沒有關閉它們，你可能會用完檔案控制代碼， 導致程式崩潰。with 陳述式完全防止了 這個問題。永遠使用 with。
MATLAB 對照：在 MATLAB 中你寫 fid = fopen('data.txt', 'r')，必須記住在 完成時呼叫 fclose(fid)。Python 的 with 陳述式就像自動的 fclose — 它會為你處理 清理工作，即使你的程式碼拋出錯誤也一樣。

編碼

預設情況下，Python 使用你系統的預設編碼。為了明確 （並避免跨平台的意外），請指定編碼： 

# 明確指定編碼 — 建議用於可攜性
with open("data.txt", "r", encoding="utf-8") as f:
    content = f.read()

這在處理包含非 ASCII 字元（帶重音的字母、中日韓字元、 表情符號等）的檔案時特別重要。

2. 寫入檔案

使用 "w" 模式寫入（會覆蓋！）

# 寫入模式 — 建立新檔案或覆蓋現有內容
with open("output.txt", "w") as f:
    f.write("Hello, World!\n")
    f.write("This is line 2.\n")
警告：使用 "w" 模式開啟檔案 會在檔案開啟的瞬間清除所有現有內容。 如果檔案包含重要資料，它就消失了。在執行腳本之前， 務必仔細檢查檔案名稱和模式。

使用 "a" 模式附加

# 附加模式 — 在末尾添加，不會清除
with open("log.txt", "a") as f:
    f.write("New log entry: everything is fine.\n")

附加模式非常適合日誌檔案、持續記錄，或任何你想要 添加資料而不破壞已有內容的情況。

.write() vs .writelines()

# .write() 接受單一字串
with open("output.txt", "w") as f:
    f.write("Line 1\n")
    f.write("Line 2\n")

# .writelines() 接受字串串列（不會自動添加換行符！）
lines = ["Line 1\n", "Line 2\n", "Line 3\n"]
with open("output.txt", "w") as f:
    f.writelines(lines)
注意：.writelines() 不會自動在項目之間添加換行字元。 你必須自己在每個字串中包含 \n。

寫入格式化資料

# 將格式化的輸出寫入檔案
students = [
    ("Alice", 92),
    ("Bob", 85),
    ("Charlie", 78),
]

with open("grades.txt", "w") as f:
    f.write("Name        Grade\n")
    f.write("-" * 20 + "\n")
    for name, grade in students:
        f.write(f"{name:<12}{grade}\n")

# grades.txt 的內容將是：
# Name        Grade
# --------------------
# Alice       92
# Bob         85
# Charlie     78
MATLAB 對照：在 MATLAB 中你會使用 fprintf(fid, '%s %d\n', name, grade)。Python 的 f-string（f"..."）有相同的用途，通常更易讀。 {name:<12} 中的 < 表示在 12 個字元的欄位中靠左對齊 — 類似 MATLAB 的 %-12s。

完整的讀取-處理-寫入範例

# 從檔案讀取數字，計算統計資料，寫入結果
with open("numbers.txt", "r") as f:
    numbers = [float(line.strip()) for line in f if line.strip()]

total = sum(numbers)
average = total / len(numbers)
maximum = max(numbers)
minimum = min(numbers)

with open("stats.txt", "w") as f:
    f.write(f"Count:   {len(numbers)}\n")
    f.write(f"Sum:     {total:.2f}\n")
    f.write(f"Average: {average:.2f}\n")
    f.write(f"Max:     {maximum:.2f}\n")
    f.write(f"Min:     {minimum:.2f}\n")

print("Statistics written to stats.txt")

3. try / except — 錯誤處理

為什麼錯誤處理很重要

當你的程式碼與外部世界互動時 — 讀取檔案、 取得使用者輸入、連接網路 — 事情可能會出錯。 檔案可能不存在。使用者可能在你期望數字時輸入字母。 網路可能斷線。沒有錯誤處理，你的程式會因為醜陋的 追蹤資訊而崩潰。有了錯誤處理，你可以優雅地回應。

基本 try / except 語法

try:
    result = 10 / 0
except ZeroDivisionError:
    print("Cannot divide by zero!")

輸出：

Cannot divide by zero!

try 內的程式碼正常執行。如果發生例外， Python 會跳到匹配的 except 區塊 而不是崩潰。

MATLAB 對照：MATLAB 使用 try / catch。Python 使用 try / except。 概念相同 — 只有關鍵字不同： 
% MATLAB
try
    result = 10 / 0;
catch ME
    disp(ME.message);
end
# Python
try:
    result = 10 / 0
except ZeroDivisionError as e:
    print(e)

捕捉特定的例外

始終捕捉特定的例外，而非使用裸露的 except:。這可以防止你意外隱藏錯誤： 

# 好 — 捕捉特定的例外
try:
    with open("config.txt", "r") as f:
        data = f.read()
except FileNotFoundError:
    print("config.txt not found. Using defaults.")

# 不好 — 裸露的 except 會捕捉所有東西（包括拼字錯誤和程式錯誤！）
try:
    with open("config.txt", "r") as f:
        data = f.read()
except:          # 不要這樣做！
    print("Something went wrong.")

多個 except 區塊

try:
    value = int(input("Enter a number: "))
    result = 100 / value
    print(f"100 / {value} = {result}")
except ValueError:
    print("That's not a valid number!")
except ZeroDivisionError:
    print("Cannot divide by zero!")
except Exception as e:
    print(f"Unexpected error: {e}")

Python 從上到下檢查每個 except 區塊， 並執行第一個匹配的。將更具體的例外放在前面， 更一般的放在後面。

else 和 finally 子句

try:
    with open("data.txt", "r") as f:
        content = f.read()
except FileNotFoundError:
    print("File not found!")
else:
    # 只在沒有發生例外時執行
    print(f"File has {len(content)} characters.")
finally:
    # 無論是否發生例外都會執行
    print("File operation attempted.")
子句 何時執行 典型用途
try 永遠（這是你「嘗試」的程式碼） 有風險的操作
except 只在 try 中發生例外時 處理錯誤
else 只在沒有例外發生時 在成功時處理結果
finally 永遠，無論如何 清理（關閉連接等）

常見的例外

例外 何時發生 範例
FileNotFoundError 檔案不存在 open("nope.txt")
ValueError 值的類型不正確 int("hello")
TypeError 操作的類型不正確 "2" + 2
ZeroDivisionError 除以零 10 / 0
IndexError 串列索引超出範圍 [1, 2, 3][5]
KeyError 字典鍵不存在 {"a": 1}["b"]
PermissionError 沒有存取檔案的權限 開啟受系統保護的檔案

何時使用 try / except（不要過度使用）

錯誤處理很強大，但不要用 try / except 包住每一行 程式碼。在以下情況使用它：

  • 你在進行檔案 I/O（檔案可能不存在，磁碟可能滿了）
  • 你在解析使用者輸入（使用者會輸入意想不到的東西）
  • 你在進行網路請求（連接可能失敗）
  • 你在轉換資料類型（int()、float() 可能失敗）

不要使用 try / except 來隱藏程式錯誤。如果你的程式碼有邏輯錯誤， 修正程式碼而不是捕捉例外然後忽略它。

實用範例：安全的檔案讀取器

def read_file_safely(filename):
    """讀取檔案並回傳其內容，如果失敗則回傳 None。"""
    try:
        with open(filename, "r", encoding="utf-8") as f:
            return f.read()
    except FileNotFoundError:
        print(f"Error: '{filename}' not found.")
        return None
    except PermissionError:
        print(f"Error: No permission to read '{filename}'.")
        return None
    except UnicodeDecodeError:
        print(f"Error: '{filename}' contains invalid UTF-8.")
        return None

# 使用方式
content = read_file_safely("config.txt")
if content is not None:
    print(f"Read {len(content)} characters.")

4. 匯入模組

什麼是模組？

模組就是一個包含 Python 程式碼的 .py 檔案 — 函式、類別、變數或可執行的陳述式。 Python 附帶了大量的標準函式庫模組， 你還可以從網路上安裝數千個額外的模組。模組讓你重複使用 程式碼而不需要複製貼上。

三種匯入方式

# 方式 1：匯入整個模組
import math
print(math.sqrt(16))    # 4.0
print(math.pi)          # 3.141592653589793

# 方式 2：匯入特定項目
from math import sqrt, pi
print(sqrt(16))         # 4.0
print(pi)               # 3.141592653589793

# 方式 3：使用別名匯入
import numpy as np      # 常見慣例
arr = np.array([1, 2, 3])
print(arr * 2)          # [2 4 6]
應該使用哪種風格？
  • import module — 最清晰。你總是能看到每個函式的來源。
  • from module import name — 當你頻繁使用某個函式，打模組前綴變得繁瑣時適用。
  • import module as alias — 使用知名的別名：np 代表 numpy，pd 代表 pandas，plt 代表 matplotlib.pyplot。

避免 from module import * — 它會將所有東西傾倒到你的命名空間中，可能導致名稱衝突。

MATLAB 對照：在 MATLAB 中，所有在路徑上的 函式都自動可用 — 不需要 import 步驟。 你使用 addpath 來添加目錄。在 Python 中， 你必須明確匯入你需要的東西。這比較冗長，但可以防止 名稱衝突並使依賴關係更加清晰。 
% MATLAB — 如果在路徑上，函式就可用
addpath('my_tools/');
result = my_function(42);

# Python — 你必須明確匯入
from my_tools import my_function
result = my_function(42)

常用的標準函式庫模組

這些模組是 Python 內建的 — 不需要安裝：

模組 用途 快速範例
os 作業系統操作（檔案、路徑、目錄） os.listdir(".")
sys 系統特定的參數和函式 sys.argv（命令列引數）
math 數學函式 math.sqrt(2)、math.log(10)
random 隨機數生成 random.randint(1, 100)
datetime 日期和時間操作 datetime.datetime.now()
json 讀寫 JSON 檔案 json.loads('{"a": 1}')
csv 讀寫 CSV 檔案 csv.reader(open("f.csv"))
pathlib 物件導向的檔案路徑（現代風格） Path("data") / "file.txt" 
# 標準函式庫模組的快速範例
import os
print(os.getcwd())              # 目前工作目錄
print(os.listdir("."))          # 列出目前目錄中的檔案

import random
print(random.randint(1, 6))     # 模擬擲骰子
print(random.choice(["A", "B", "C"]))  # 隨機選取一個項目

import datetime
now = datetime.datetime.now()
print(now.strftime("%Y-%m-%d %H:%M"))  # 例如 "2026-03-31 14:30"

import json
data = {"name": "Alice", "score": 95}
json_string = json.dumps(data, indent=2)
print(json_string)

from pathlib import Path
p = Path("data") / "results" / "output.txt"
print(p)  # data/results/output.txt（跨平台！）

使用 pip 安裝第三方模組

標準函式庫很豐富，但 Python 真正的力量來自 數十萬個第三方套件。你可以使用 pip （Python 的套件管理器）來安裝它們： 

# 在你的終端機中（不是在 Python 裡面）：
pip install requests       # HTTP 函式庫
pip install pandas         # 資料分析
pip install matplotlib     # 繪圖
pip install numpy          # 數值計算
pip install flask          # 網頁框架
# 然後在你的 Python 程式碼中：
import requests
response = requests.get("https://api.github.com")
print(response.status_code)   # 200
print(response.json())        # 解析後的 JSON 資料
MATLAB 對照：Python 的 pip 類似 MATLAB 的附加元件探索器或 matlab.addons.install() 函式，但 pip 可以存取 超過 500,000 個免費套件。MATLAB 的工具箱很強大但 價格昂貴 — 每個都需要單獨的授權。Python 套件是免費的，由社群維護。

__name__ == "__main__" 是什麼意思？

你會經常在 Python 腳本的底部看到這個模式： 

# my_script.py

def greet(name):
    return f"Hello, {name}!"

def main():
    print(greet("World"))

if __name__ == "__main__":
    main()

以下是正在發生的事情：

  • 每個 Python 檔案都有一個內建變數叫做 __name__。
  • 當你直接執行一個檔案時（例如 python my_script.py），Python 會將 __name__ 設定為 "__main__"。
  • 當你將一個檔案作為模組匯入時（例如 import my_script），Python 會將 __name__ 設定為模組的名稱 （"my_script"）。

if __name__ == "__main__": 守衛的意思是： 「只有在我被直接執行時才運行這段程式碼，而不是被匯入時。」 這讓同一個檔案既可以作為可重複使用的模組， 又可以作為獨立的腳本。 

# 沒有守衛的話，匯入時會不小心印出 "Hello, World!"。
# 守衛防止了這種情況。

# 直接執行：    python my_script.py  → 印出 "Hello, World!"
# 匯入它：      import my_script     → 什麼都不會發生（好！）

5. 建立你自己的模組

任何 .py 檔案都是模組

建立模組就像儲存一個 .py 檔案一樣簡單。 假設你建立了一個名為 helpers.py 的檔案： 

# helpers.py — 你的自訂模組

def celsius_to_fahrenheit(c):
    """將攝氏轉換為華氏。"""
    return c * 9 / 5 + 32

def fahrenheit_to_celsius(f):
    """將華氏轉換為攝氏。"""
    return (f - 32) * 5 / 9

PI = 3.14159265358979

現在在同一目錄中的另一個檔案裡，你可以匯入並使用它： 

# main.py — 使用 helpers 模組

from helpers import celsius_to_fahrenheit, PI

temp_f = celsius_to_fahrenheit(100)
print(f"100 C = {temp_f} F")     # 100 C = 212.0 F
print(f"PI = {PI}")              # PI = 3.14159265358979
# 或匯入整個模組
import helpers

temp_c = helpers.fahrenheit_to_celsius(72)
print(f"72 F = {temp_c:.1f} C")  # 72 F = 22.2 C

在你自己的模組中使用 __name__ 守衛

添加守衛使你的模組既可匯入又可執行： 

# helpers.py — 帶有 __name__ 守衛

def celsius_to_fahrenheit(c):
    return c * 9 / 5 + 32

def fahrenheit_to_celsius(f):
    return (f - 32) * 5 / 9

if __name__ == "__main__":
    # 這段只在你執行 python helpers.py 時運行
    # 當別人執行 import helpers 時不會運行
    print("Running tests...")
    assert celsius_to_fahrenheit(0) == 32
    assert celsius_to_fahrenheit(100) == 212
    assert fahrenheit_to_celsius(32) == 0
    print("All tests passed!")

這是常見的模式：將測試程式碼或示範程式碼放在 __name__ 守衛內，讓檔案的其餘部分 保持乾淨以供匯入。

組織小型專案

以下是典型的小型 Python 專案可能的結構： 

my_project/
    main.py          # 入口點 — 執行這個
    helpers.py       # 工具函式
    file_io.py       # 檔案讀寫函式
    data/
        input.txt
        output.txt
# main.py

def read_temperatures(filename):
    """讀取溫度檔案（每行一個）並回傳浮點數串列。"""
    with open(filename, "r") as f:
        return [float(line.strip()) for line in f if line.strip()]

def write_results(filename, temperatures):
    """將溫度寫入檔案，每行一個。"""
    with open(filename, "w") as f:
        for temp in temperatures:
            f.write(f"{temp:.1f}\n")

# analyze_scores.py — 一個完整的迷你程式

import os
from datetime import datetime

def load_scores(filename):
    """從檔案載入學生分數。每行格式：'Name,Score'"""
    scores = {}
    try:
        with open(filename, "r", encoding="utf-8") as f:
            for line_num, line in enumerate(f, start=1):
                line = line.strip()
                if not line:
                    continue
                try:
                    name, score = line.split(",")
                    scores[name.strip()] = float(score.strip())
                except ValueError:
                    print(f"Warning: skipping malformed line {line_num}: '{line}'")
    except FileNotFoundError:
        print(f"Error: '{filename}' not found.")
    return scores

def compute_stats(scores):
    """從分數字典計算基本統計資料。"""
    if not scores:
        return None
    values = list(scores.values())
    return {
        "count": len(values),
        "average": sum(values) / len(values),
        "highest": max(values),
        "lowest": min(values),
        "top_student": max(scores, key=scores.get),
    }

def write_report(filename, stats):
    """將統計報告寫入檔案。"""
    with open(filename, "w", encoding="utf-8") as f:
        f.write(f"Report generated: {datetime.now().strftime('%Y-%m-%d %H:%M')}\n")
        f.write("=" * 40 + "\n")
        f.write(f"Students:     {stats['count']}\n")
        f.write(f"Average:      {stats['average']:.1f}\n")
        f.write(f"Highest:      {stats['highest']:.1f}\n")
        f.write(f"Lowest:       {stats['lowest']:.1f}\n")
        f.write(f"Top student:  {stats['top_student']}\n")
    print(f"Report written to '{filename}'.")

if __name__ == "__main__":
    scores = load_scores("scores.csv")
    if scores:
        stats = compute_stats(scores)
        write_report("report.txt", stats)
    else:
        print("No scores to analyse.")

練習 6：帶錯誤處理的檔案複製器

撰寫一個函式 copy_file(source, destination)， 將一個文字檔案的內容複製到另一個。處理來源檔案不存在和 權限問題的錯誤。

顯示解答 
def copy_file(source, destination):
    """將文字內容從來源複製到目的地。"""
    try:
        with open(source, "r", encoding="utf-8") as src:
            content = src.read()
    except FileNotFoundError:
        print(f"Error: source file '{source}' not found.")
        return False
    except PermissionError:
        print(f"Error: no permission to read '{source}'.")
        return False

    try:
        with open(destination, "w", encoding="utf-8") as dst:
            dst.write(content)
    except PermissionError:
        print(f"Error: no permission to write to '{destination}'.")
        return False

    print(f"Copied '{source}' to '{destination}' ({len(content)} characters).")
    return True

# 使用方式
copy_file("original.txt", "backup.txt")

練習 7：CSV 讀取器與平均計算器

撰寫一個程式，讀取一個 CSV 檔案，其中每行包含 學生姓名和成績（例如 Alice,92），計算 班級平均分數，並將摘要寫入新檔案。

顯示解答 
def process_grades(input_file, output_file):
    students = []

    try:
        with open(input_file, "r", encoding="utf-8") as f:
            for line in f:
                line = line.strip()
                if not line:
                    continue
                try:
                    name, grade = line.split(",")
                    students.append((name.strip(), float(grade.strip())))
                except ValueError:
                    print(f"Skipping invalid line: '{line}'")
    except FileNotFoundError:
        print(f"File '{input_file}' not found.")
        return

    if not students:
        print("No valid student data found.")
        return

    grades = [g for _, g in students]
    average = sum(grades) / len(grades)

    with open(output_file, "w", encoding="utf-8") as f:
        f.write("Student Grade Report\n")
        f.write("=" * 30 + "\n")
        for name, grade in sorted(students, key=lambda x: x[1], reverse=True):
            status = "PASS" if grade >= 60 else "FAIL"
            f.write(f"{name:<15} {grade:6.1f}  {status}\n")
        f.write("=" * 30 + "\n")
        f.write(f"Class average: {average:.1f}\n")
        f.write(f"Highest: {max(grades):.1f}\n")
        f.write(f"Lowest:  {min(grades):.1f}\n")

    print(f"Report written to '{output_file}'.")

process_grades("students.csv", "grade_report.txt")
