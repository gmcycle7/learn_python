專案 1：簡易計算機

目標

建構一個互動式計算機，要求使用者輸入兩個數字和一個算術運算符（+、-、*、/），執行計算並印出結果。程式能優雅地處理無效輸入而不會當機。

使用的概念

函式 條件判斷 input() try / except while 迴圈 f-strings

完整程式碼

# ============================================================
# simple_calculator.py — 互動式算術計算機
# ============================================================

def add(a, b):
    """回傳 a 和 b 的和。"""
    return a + b

def subtract(a, b):
    """回傳 a 和 b 的差。"""
    return a - b

def multiply(a, b):
    """回傳 a 和 b 的積。"""
    return a * b

def divide(a, b):
    """回傳 a 除以 b 的商。
    如果 b 為零則引發 ValueError。"""
    if b == 0:
        raise ValueError("Cannot divide by zero!")
    return a / b

def get_number(prompt):
    """要求使用者輸入一個數字。持續詢問直到輸入有效。"""
    while True:
        try:
            return float(input(prompt))
        except ValueError:
            print("  Invalid input. Please enter a numeric value.")

def main():
    """主迴圈 — 運行計算機直到使用者退出。"""
    print("=" * 40)
    print("  Simple Python Calculator")
    print("=" * 40)

    # 將運算符號對應到函式
    operations = {
        "+": add,
        "-": subtract,
        "*": multiply,
        "/": divide,
    }

    while True:
        # --- 取得第一個數字 ---
        num1 = get_number("\nEnter the first number: ")

        # --- 取得運算符 ---
        print("Choose an operation: +, -, *, /")
        op = input("Enter operator: ").strip()

        if op not in operations:
            print(f"  Unknown operator '{op}'. Try again.")
            continue  # 重新開始迴圈

        # --- 取得第二個數字 ---
        num2 = get_number("Enter the second number: ")

        # --- 執行計算 ---
        try:
            result = operations[op](num1, num2)
            print(f"\n  Result: {num1} {op} {num2} = {result}")
        except ValueError as e:
            print(f"\n  Error: {e}")

        # --- 詢問是否繼續 ---
        again = input("\nAnother calculation? (y/n): ").strip().lower()
        if again != "y":
            print("Goodbye!")
            break

if __name__ == "__main__":
    main()

逐塊解說

1. 算術函式（第 5–22 行）

每個運算被包裝在各自的函式中（add、subtract、multiply、divide）。這比把所有邏輯放在一個大的 if/elif 鏈中更乾淨，並且使每個運算可以獨立測試。divide 函式明確檢查除以零的情況，並引發帶有清晰訊息的 ValueError。

2. 輸入輔助函式 — get_number()（第 24–29 行）

這個函式使用 while True 迴圈結合 try/except。如果使用者輸入無法轉換為 float 的內容（例如 "abc"），ValueError 會被捕獲，並再次提示使用者。迴圈只有在回傳有效數字時才會退出。

3. 運算分派字典（第 39–44 行）

我們不為每個運算符寫 if op == "+": ...，而是將函式儲存在以運算符字串為鍵的字典中。這是常見的 Python 模式 — 分派表 — 取代冗長的條件鏈。

MATLAB 對照：在 MATLAB 中你可能會使用 switch 語句。Python 沒有 switch（雖然 Python 3.10+ 有 match/case）。字典分派模式是慣用的 Python 方法，既簡潔又易於擴展。

4. 主迴圈（第 47–68 行）

while True 迴圈讓計算機持續運行，直到使用者輸入 y 以外的內容。在迴圈中我們收集輸入、查找運算、用兩個數字呼叫它，並印出結果。計算呼叫周圍的 try/except 捕獲除以零的錯誤。

5. 入口點守衛（第 70 行）

if __name__ == "__main__": 確保 main() 只在腳本直接執行時運行，而不是被作為模組匯入時。這是一個 Python 最佳實踐，你會在每個結構良好的腳本中看到。

如何執行

# 將程式碼儲存為 simple_calculator.py，然後：
python simple_calculator.py

程式會在終端機中互動式地提示你。在被詢問時輸入數字和運算符。

修改與延伸

  • 新增更多運算：次方（**）、取餘數（%）、整數除法（//）。
  • 新增歷史記錄串列，儲存過去的計算並在需要時印出。
  • 支援像 "3 + 5" 這樣在同一行輸入的運算式（使用 str.split()）。
  • 將結果寫入日誌檔，以便記錄所有計算。

專案 2：檔案整理器

目標

根據副檔名自動將資料夾中的檔案分類到子資料夾。例如，.txt 檔案移到 TextFiles 資料夾，.jpg 和 .png 檔案移到 Images 資料夾，依此類推。不在對應表中的副檔名會移到 Misc 資料夾。

使用的概念

os shutil pathlib 字典 迴圈 條件判斷 檔案 I/O

完整程式碼

# ============================================================
# file_organizer.py — 按副檔名整理資料夾中的檔案
# ============================================================

import os
import shutil
from pathlib import Path

# ----- 設定 -----
# 將每個副檔名（小寫）對應到目標子資料夾名稱。
# 根據需要新增或移除項目。
EXTENSION_MAP = {
    # 文件
    ".txt":  "TextFiles",
    ".pdf":  "Documents",
    ".docx": "Documents",
    ".xlsx": "Spreadsheets",
    ".csv":  "Spreadsheets",
    # 圖片
    ".jpg":  "Images",
    ".jpeg": "Images",
    ".png":  "Images",
    ".gif":  "Images",
    ".bmp":  "Images",
    # 程式碼 / 腳本
    ".py":   "PythonScripts",
    ".m":    "MatlabFiles",
    ".js":   "Code",
    ".html": "Code",
    ".css":  "Code",
    # 資料
    ".json": "Data",
    ".xml":  "Data",
    # 壓縮檔
    ".zip":  "Archives",
    ".tar":  "Archives",
    ".gz":   "Archives",
}

# 未列出副檔名的備用資料夾
DEFAULT_FOLDER = "Misc"


def organize(folder_path):
    """將 *folder_path* 中的每個檔案按副檔名移到子資料夾。

    Parameters
    ----------
    folder_path : str or Path
        你想要整理的目錄。
    """
    source = Path(folder_path)

    # 安全檢查 — 確保路徑存在且是目錄
    if not source.is_dir():
        print(f"Error: '{source}' is not a valid directory.")
        return

    moved_count = 0  # 追蹤移動了多少檔案

    # 遍歷資料夾中的每個項目
    for item in source.iterdir():
        # 跳過目錄 — 我們只想移動檔案
        if item.is_dir():
            continue

        # 取得小寫的副檔名（例如 ".TXT" -> ".txt"）
        ext = item.suffix.lower()

        # 查找目標資料夾；找不到則使用 DEFAULT_FOLDER
        target_name = EXTENSION_MAP.get(ext, DEFAULT_FOLDER)

        # 建構目標子資料夾的完整路徑
        target_dir = source / target_name

        # 如果子資料夾不存在則建立
        target_dir.mkdir(exist_ok=True)

        # 建構目標檔案路徑
        destination = target_dir / item.name

        # 處理名稱衝突：如果目標資料夾中已存在同名檔案，
        # 加上數字後綴。
        counter = 1
        while destination.exists():
            stem = item.stem        # 不含副檔名的檔名
            destination = target_dir / f"{stem}_{counter}{ext}"
            counter += 1

        # 移動檔案
        shutil.move(str(item), str(destination))
        print(f"  Moved: {item.name}  ->  {target_name}/")
        moved_count += 1

    # 摘要
    print(f"\nDone! Moved {moved_count} file(s).")


if __name__ == "__main__":
    # 詢問使用者要整理哪個資料夾
    folder = input("Enter the path of the folder to organize: ").strip()

    # 如果使用者按 Enter 沒有輸入，使用目前目錄
    if not folder:
        folder = "."

    print(f"\nOrganizing files in: {Path(folder).resolve()}\n")
    organize(folder)

逐塊解說

1. 匯入（第 5–7 行）

os 提供基本的作業系統互動，shutil 提供 move() 函式用於跨目錄移動檔案，而 pathlib.Path 提供物件導向的方式來操作檔案路徑（比用 os.path.join 串接字串乾淨多了）。

2. 副檔名對應字典（第 12–39 行）

一個字典將小寫副檔名對應到資料夾名稱。使用字典使得對應表易於閱讀和修改。第 72 行的 .get(ext, DEFAULT_FOLDER) 呼叫會回傳對應的資料夾，如果副檔名未知則回傳 "Misc"。

3. organize() 函式（第 43–92 行）

這是核心邏輯。source.iterdir() 產生目錄中的每個項目。我們跳過子目錄、查找每個檔案的副檔名、用 mkdir(exist_ok=True) 建立目標子資料夾（如果已存在不會報錯）、用 while 迴圈處理名稱衝突，最後呼叫 shutil.move()。

提示：mkdir() 上的 exist_ok=True 參數是 Python 版的「如果不存在則建立」。不加的話，對已存在的目錄呼叫 mkdir() 會引發 FileExistsError。

4. 入口點（第 95–103 行）

提示使用者輸入資料夾路徑，如果沒有輸入則預設為目前目錄，然後呼叫 organize()。

MATLAB 對照：在 MATLAB 中你可能會用 dir() 列出檔案和 movefile() 移動它們。Python 的 pathlib 類似 MATLAB 的 fullfile() 用於建構路徑，但它也提供 .suffix、.stem 和 .name 作為方便的屬性。

如何執行

# 儲存為 file_organizer.py，然後：
python file_organizer.py

# 提示時輸入資料夾路徑（例如 C:\Users\you\Downloads）
# 或按 Enter 整理目前目錄。
警告：這個腳本會移動檔案。請先在包含副本的資料夾上測試，不要在你唯一的重要資料上測試。

修改與延伸

  • 新增 --dry-run 旗標，只印出將會發生什麼而不實際移動檔案。
  • 新增 --undo 功能，將移動記錄寫入日誌以便復原。
  • 讓副檔名對應表可透過 JSON 檔案配置。
  • 新增遞迴整理功能（也整理子資料夾中的檔案）。

專案 3：CSV 分析器

目標

讀取一個 CSV 檔案，識別數值欄位，並為每個欄位計算基本統計資料：計數、平均值、最小值和最大值。在終端機印出乾淨的摘要表格。這個專案只使用標準函式庫 — 不需要 pandas 或 numpy。

使用的概念

csv 模組 檔案 I/O 字典 串列 數學（基礎） 字串格式化

完整程式碼

# ============================================================
# csv_analyzer.py — 讀取 CSV 並計算基本統計資料
# ============================================================

import csv
import sys


def read_csv(filepath):
    """讀取 CSV 檔案並回傳 (headers, rows)。

    Parameters
    ----------
    filepath : str
        CSV 檔案的路徑。

    Returns
    -------
    headers : list of str
        第一列的欄位名稱。
    rows : list of list of str
        所有資料列（每列是字串值的串列）。
    """
    with open(filepath, newline="", encoding="utf-8") as f:
        reader = csv.reader(f)
        headers = next(reader)          # 第一列 = 欄位名稱
        rows = [row for row in reader]   # 其餘列 = 資料
    return headers, rows


def is_numeric(value):
    """如果 *value* 可以轉換為 float 則回傳 True。"""
    try:
        float(value)
        return True
    except (ValueError, TypeError):
        return False


def compute_stats(values):
    """給定一個 float 串列，回傳含 count、mean、min、max 的字典。

    Parameters
    ----------
    values : list of float

    Returns
    -------
    dict with keys: 'count', 'mean', 'min', 'max'
    """
    n = len(values)
    if n == 0:
        return {"count": 0, "mean": None, "min": None, "max": None}

    total = sum(values)
    return {
        "count": n,
        "mean": total / n,
        "min": min(values),
        "max": max(values),
    }


def analyze(filepath):
    """讀取 *filepath* 的 CSV 並印出數值欄位的統計資料。"""
    headers, rows = read_csv(filepath)

    # 對每個欄位收集數值
    # 鍵：欄位索引，值：float 串列
    numeric_data = {}

    for col_idx, header in enumerate(headers):
        col_values = []
        for row in rows:
            # 某些列可能欄位較少；優雅地跳過
            if col_idx < len(row) and is_numeric(row[col_idx]):
                col_values.append(float(row[col_idx]))
        # 只保留至少有一個數值的欄位
        if col_values:
            numeric_data[col_idx] = col_values

    if not numeric_data:
        print("No numeric columns found in the CSV.")
        return

    # ----- 印出報告 -----
    print(f"\nFile: {filepath}")
    print(f"Total rows (excluding header): {len(rows)}")
    print(f"Total columns: {len(headers)}")
    print(f"Numeric columns found: {len(numeric_data)}\n")

    # 表頭
    print(f"{'Column':<20} {'Count':>8} {'Mean':>12} {'Min':>12} {'Max':>12}")
    print("-" * 66)

    for col_idx, values in numeric_data.items():
        stats = compute_stats(values)
        name = headers[col_idx]
        print(
            f"{name:<20} "
            f"{stats['count']:>8} "
            f"{stats['mean']:>12.4f} "
            f"{stats['min']:>12.4f} "
            f"{stats['max']:>12.4f}"
        )

    print()


if __name__ == "__main__":
    # 從命令列接受檔案路徑或提示輸入
    if len(sys.argv) > 1:
        csv_path = sys.argv[1]
    else:
        csv_path = input("Enter the path to the CSV file: ").strip()

    analyze(csv_path)

逐塊解說

1. read_csv()（第 9–30 行）

使用 Python 內建的 csv.reader 開啟 CSV 檔案。第一列用 next(reader) 單獨讀取以取得欄位標題。其餘列用串列推導式收集成串列的串列。

2. is_numeric()（第 33–39 行）

一個小型輔助函式，嘗試將值轉換為 float。如果轉換失敗，則該值不是數值。這是標準的 Python「請求寬恕而非許可」（EAFP）模式。

3. compute_stats()（第 42–60 行）

接受一個 float 串列並回傳 count、mean、min 和 max 作為字典。使用內建的 sum()、min() 和 max() 函式。不需要外部函式庫。

MATLAB 對照：在 MATLAB 中你會直接在欄向量上呼叫 mean()、min()、max() 和 numel()。Python 的內建函式可用於任何可迭代物件（串列、元組等），而不需要陣列。

4. analyze()（第 63–101 行）

主要的分析函式。它讀取 CSV、遍歷每個欄位以收集數值、為每個數值欄位計算統計資料，並使用 f-string 對齊指定符如 <20（左對齊，20 字元）和 >12.4f（右對齊，12 字元，4 位小數）印出格式化的表格。

5. 入口點（第 104–109 行）

接受 CSV 路徑作為命令列引數，或在沒有引數時互動式提示輸入。

專案 4：簡易圖表產生器

目標

產生樣本資料並建立兩個出版品質的圖表：一個顯示趨勢的折線圖，以及一個比較類別的長條圖。使用 savefig() 將兩個圖表儲存為影像檔。

使用的概念

matplotlib 串列 串列推導式 math 模組 savefig() 子圖

完整程式碼

# ============================================================
# plot_generator.py — 建立折線圖和長條圖
# ============================================================

import math
import matplotlib.pyplot as plt


def generate_line_plot(filename="line_plot.png"):
    """建立並儲存正弦波訊號的折線圖。

    這模擬了隨時間繪製實驗資料的情境 — 對從
    MATLAB 轉來的人而言是非常常見的任務。
    """
    # --- 產生資料 ---
    # 從 0 到 4*pi 的 100 個等距 x 值
    num_points = 100
    x = [i * (4 * math.pi) / (num_points - 1) for i in range(num_points)]

    # 兩個訊號：正弦波和衰減正弦波
    y_sine    = [math.sin(xi) for xi in x]
    y_damped  = [math.exp(-0.2 * xi) * math.sin(xi) for xi in x]

    # --- 建立圖表 ---
    fig, ax = plt.subplots(figsize=(8, 4))

    ax.plot(x, y_sine,   label="sin(x)",                  color="steelblue", linewidth=2)
    ax.plot(x, y_damped, label="exp(-0.2x) * sin(x)",     color="tomato",    linewidth=2, linestyle="--")

    # --- 標籤和樣式 ---

目標

讀取日誌檔、擷取所有包含錯誤訊息的行、計算每個唯一錯誤出現的次數，並將摘要報告寫入新檔案。這是 Python 擅長的文字處理任務。

使用的概念

檔案 I/O 字串方法 字典 排序 迴圈 f-strings

完整程式碼

# ============================================================
# log_parser.py — 解析日誌檔並摘要錯誤
# ============================================================

import sys
from datetime import datetime


def parse_log(filepath):
    """讀取日誌檔並擷取錯誤行。

    假設每行日誌的格式為：
        [TIMESTAMP] LEVEL: message
    例如：
        [2025-01-15 10:23:45] ERROR: Connection timed out

    Returns
    -------
    error_lines : list of str
        每個錯誤行的完整文字。
    error_counts : dict
        將每個唯一錯誤訊息對應到出現次數。
    total_lines : int
        檔案中的總行數。
    """
    error_lines = []
    error_counts = {}
    total_lines = 0

    with open(filepath, "r", encoding="utf-8") as f:
        for line in f:
            total_lines += 1
            stripped = line.strip()

            # 檢查這行是否包含 ERROR 級別訊息
            if "ERROR" in stripped.upper():
                error_lines.append(stripped)

                # 擷取訊息部分（"ERROR:" 之後的內容）
                # 如果存在 "ERROR:" 則取其後的所有內容；否則使用整行
                if "ERROR:" in stripped:
                    msg = stripped.split("ERROR:", 1)[1].strip()
                elif "error:" in stripped:
                    msg = stripped.split("error:", 1)[1].strip()
                else:
                    msg = stripped  # 備用：使用整行

                # 計算此確切訊息的出現次數
                error_counts[msg] = error_counts.get(msg, 0) + 1

    return error_lines, error_counts, total_lines


def write_report(error_lines, error_counts, total_lines, log_path, report_path):
    """將錯誤摘要報告寫入 *report_path*。

    Parameters
    ----------
    error_lines : list of str
    error_counts : dict
    total_lines : int
    log_path : str
    report_path : str
    """
    # 按次數排序錯誤，最多的在前
    sorted_errors = sorted(error_counts.items(), key=lambda x: x[1], reverse=True)

    with open(report_path, "w", encoding="utf-8") as f:
        f.write("=" * 60 + "\n")
        f.write("  LOG ERROR SUMMARY REPORT\n")
        f.write(f"  Generated: {datetime.now().strftime('%Y-%m-%d %H:%M:%S')}\n")
        f.write("=" * 60 + "\n\n")

        f.write(f"Source log file: {log_path}\n")
        f.write(f"Total lines in log: {total_lines}\n")
        f.write(f"Total error lines:  {len(error_lines)}\n")
        f.write(f"Unique errors:      {len(error_counts)}\n\n")

        # --- 錯誤頻率表 ---
        f.write("-" * 60 + "\n")
        f.write(f"{'Count':>6}  {'Error Message'}\n")
        f.write("-" * 60 + "\n")
        for msg, count in sorted_errors:
            f.write(f"{count:>6}  {msg}\n")
        f.write("-" * 60 + "\n\n")

        # --- 完整錯誤行 ---
        f.write("FULL ERROR LINES:\n\n")
        for i, line in enumerate(error_lines, start=1):
            f.write(f"  [{i}] {line}\n")
        f.write("\n" + "=" * 60 + "\n")
        f.write("  END OF REPORT\n")
        f.write("=" * 60 + "\n")


def main():
    """解析日誌檔並產生報告。"""
    # 取得檔案路徑
    if len(sys.argv) > 1:
        log_path = sys.argv[1]
    else:
        log_path = input("Enter path to the log file: ").strip()

    report_path = log_path.rsplit(".", 1)[0] + "_error_report.txt"

    print(f"Parsing: {log_path}")
    error_lines, error_counts, total_lines = parse_log(log_path)

    if not error_lines:
        print("No errors found in the log file.")
        return

    # 寫入報告
    write_report(error_lines, error_counts, total_lines, log_path, report_path)

    # 同時在終端機印出快速摘要
    print(f"\nTotal lines:  {total_lines}")
    print(f"Error lines:  {len(error_lines)}")
    print(f"Unique errors: {len(error_counts)}")
    print(f"\nTop errors:")
    for msg, count in sorted(error_counts.items(), key=lambda x: x[1], reverse=True)[:5]:
        print(f"  {count:>4}x  {msg}")
    print(f"\nFull report written to: {report_path}")


if __name__ == "__main__":
    main()

專案 6：批次檔案重新命名器

目標

根據模式重新命名資料夾中的所有檔案：加上前綴、序號，以及可選的當前日期。例如，將 photo.jpg 重新命名為 vacation_001_2025-01-15_photo.jpg。

使用的概念

os pathlib datetime f-strings enumerate() 排序

完整程式碼

# ============================================================
# batch_renamer.py — 以前綴、編號和日期重新命名檔案
# ============================================================

import os
from pathlib import Path
from datetime import datetime


def preview_renames(folder_path, prefix="file", include_date=True, start_num=1,
                    num_digits=3, extensions=None):
    """建立 (old_name, new_name) 配對的串列，不實際重新命名。

    Parameters
    ----------
    folder_path : str or Path
        包含要重新命名檔案的目錄。
    prefix : str
        要加在每個檔名前面的字串。
    include_date : bool
        是否在新名稱中包含今天的日期。
    start_num : int
        序號計數器的起始數字。
    num_digits : int
        將計數器補零到此位數（例如 3 -> 001, 002）。
    extensions : set or None
        如果提供，只重新命名具有這些副檔名的檔案（例如 {".jpg", ".png"}）。
        如果為 None，重新命名所有檔案。

    Returns
    -------
    rename_pairs : list of (Path, Path)
        每個元組是 (old_path, new_path)。
    """
    source = Path(folder_path)
    if not source.is_dir():
        print(f"Error: '{source}' is not a valid directory.")
        return []

    # 取得日期字串（例如 "2025-01-15"）
    date_str = datetime.now().strftime("%Y-%m-%d") if include_date else ""

    # 收集並排序檔案（按名稱排序以保持一致的順序）
    files = sorted(
        [f for f in source.iterdir() if f.is_file()],
        key=lambda f: f.name.lower()
    )

    # 如果指定了副檔名則篩選
    if extensions:
        extensions = {ext.lower() for ext in extensions}
        files = [f for f in files if f.suffix.lower() in extensions]

    rename_pairs = []
    for i, filepath in enumerate(files, start=start_num):
        # 建構新檔名
        number_part = str(i).zfill(num_digits)     # 例如 "001"

        if include_date:
            new_name = f"{prefix}_{number_part}_{date_str}{filepath.suffix}"
        else:
            new_name = f"{prefix}_{number_part}{filepath.suffix}"

        new_path = filepath.parent / new_name
        rename_pairs.append((filepath, new_path))

    return rename_pairs


def execute_renames(rename_pairs):
    """執行實際的檔案重新命名。

    Parameters
    ----------
    rename_pairs : list of (Path, Path)
    """
    for old_path, new_path in rename_pairs:
        # 安全措施：不覆蓋已存在的檔案
        if new_path.exists():
            print(f"  SKIP (already exists): {new_path.name}")
            continue
        os.rename(str(old_path), str(new_path))
        print(f"  Renamed: {old_path.name}  ->  {new_path.name}")


def main():
    """互動式批次檔案重新命名器。"""
    print("=" * 50)
    print("  Batch File Renamer")
    print("=" * 50)

    folder = input("\nFolder to rename files in: ").strip() or "."
    prefix = input("Prefix for new filenames [file]: ").strip() or "file"

    date_choice = input("Include date in filename? (y/n) [y]: ").strip().lower()
    include_date = date_choice != "n"

    ext_input = input("Filter by extensions (comma-separated, e.g. .jpg,.png) or Enter for all: ").strip()
    extensions = None
    if ext_input:
        extensions = {e.strip() for e in ext_input.split(",")}

    # 產生預覽
    rename_pairs = preview_renames(folder, prefix, include_date, extensions=extensions)

    if not rename_pairs:
        print("\nNo files to rename.")
        return

    # 顯示預覽
    print(f"\n--- Preview ({len(rename_pairs)} files) ---\n")
    for old_path, new_path in rename_pairs:
        print(f"  {old_path.name}  ->  {new_path.name}")

    # 確認
    confirm = input(f"\nProceed with renaming {len(rename_pairs)} files? (y/n): ").strip().lower()
    if confirm == "y":
        print()
        execute_renames(rename_pairs)
        print(f"\nDone! Renamed {len(rename_pairs)} file(s).")
    else:
        print("Cancelled. No files were renamed.")


if __name__ == "__main__":
    main()

專案 7：資料夾報告產生器

目標

掃描整個目錄樹（包括所有子目錄）並產生報告，顯示：檔案總數、總大小、按副檔名分組的檔案數量和大小，以及最大的檔案。這對了解磁碟空間使用情況很有用。

使用的概念

os.walk() pathlib 字典 排序 字串格式化 函式

完整程式碼

# ============================================================
# folder_report.py — 掃描目錄樹並產生報告
# ============================================================

import os
import sys
from pathlib import Path
from datetime import datetime


def format_size(size_bytes):
    """將位元組大小轉換為人類可讀的字串。

    範例：1024 -> '1.00 KB', 1048576 -> '1.00 MB'
    """
    for unit in ["B", "KB", "MB", "GB", "TB"]:
        if size_bytes < 1024:
            return f"{size_bytes:.2f} {unit}"
        size_bytes /= 1024
    return f"{size_bytes:.2f} PB"


def scan_directory(root_path):
    """遍歷目錄樹並收集檔案資訊。

    Parameters
    ----------
    root_path : str or Path
        要掃描的最上層目錄。

    Returns
    -------
    file_list : list of dict
        每個字典含鍵：'path', 'name', 'ext', 'size'。
    dir_count : int
        找到的子目錄數量。
    """
    root = Path(root_path)
    file_list = []
    dir_count = 0

    for dirpath, dirnames, filenames in os.walk(root):
        dir_count += len(dirnames)

        for fname in filenames:
            full_path = Path(dirpath) / fname
            try:
                size = full_path.stat().st_size
            except (OSError, PermissionError):
                # 跳過無法存取的檔案
                continue

            ext = full_path.suffix.lower() if full_path.suffix else "(no extension)"

            file_list.append({
                "path": str(full_path),
                "name": fname,
                "ext": ext,
                "size": size,
            })

    return file_list, dir_count


def generate_report(root_path, file_list, dir_count, top_n=10):
    """產生並印出格式化的報告。

    Parameters
    ----------
    root_path : str
    file_list : list of dict
    dir_count : int
    top_n : int
        要顯示的最大檔案數量。
    """
    total_size = sum(f["size"] for f in file_list)
    total_files = len(file_list)

    # --- 按副檔名分組 ---
    ext_stats = {}  # ext -> {"count": int, "size": int}
    for f in file_list:
        ext = f["ext"]
        if ext not in ext_stats:
            ext_stats[ext] = {"count": 0, "size": 0}
        ext_stats[ext]["count"] += 1
        ext_stats[ext]["size"] += f["size"]

    # 按總大小排序副檔名（降序）
    sorted_exts = sorted(ext_stats.items(), key=lambda x: x[1]["size"], reverse=True)

    # --- 最大的檔案 ---
    largest_files = sorted(file_list, key=lambda x: x["size"], reverse=True)[:top_n]

    # --- 建構報告字串 ---
    lines = []
    lines.append("=" * 70)
    lines.append("  FOLDER REPORT")
    lines.append(f"  Generated: {datetime.now().strftime('%Y-%m-%d %H:%M:%S')}")
    lines.append("=" * 70)
    lines.append("")
    lines.append(f"  Root directory:   {root_path}")
    lines.append(f"  Total files:      {total_files:,}")
    lines.append(f"  Total folders:    {dir_count:,}")
    lines.append(f"  Total size:       {format_size(total_size)}")
    lines.append("")

    # --- 副檔名明細 ---
    lines.append("-" * 70)
    lines.append(f"  {'Extension':<18} {'Files':>8} {'Total Size':>14} {'% of Total':>12}")
    lines.append("-" * 70)
    for ext, stats in sorted_exts:
        pct = (stats["size"] / total_size * 100) if total_size > 0 else 0
        lines.append(
            f"  {ext:<18} {stats['count']:>8,} {format_size(stats['size']):>14} {pct:>11.1f}%"
        )
    lines.append("-" * 70)
    lines.append("")

    # --- 最大的檔案 ---
    lines.append(f"  TOP {top_n} LARGEST FILES:")
    lines.append("")
    for i, f in enumerate(largest_files, start=1):

# （不需要 numpy，但 numpy 會簡化許多操作。）

import math
import random
import matplotlib.pyplot as plt


# ----- 資料產生 -----

def generate_data(n=50, slope=2.5, intercept=10, noise_level=5.0, seed=42):
    """沿著一條直線產生 n 個帶高斯雜訊的資料點。

    y = slope * x + intercept + noise

    Parameters
    ----------
    n : int            資料點數量。
    slope : float      直線的真實斜率。
    intercept : float  真實的 y 截距。
    noise_level : float  新增雜訊的標準差。
    seed : int         隨機種子以確保可重現性。

    Returns
    -------
    x_data : list of float
    y_data : list of float
    """
    random.seed(seed)
    x_data = [i * 1.0 for i in range(n)]
    y_data = [slope * x + intercept + random.gauss(0, noise_level) for x in x_data]
    return x_data, y_data


# ----- 統計 -----

def calc_mean(values):
    """回傳數字串列的算術平均值。"""
    return sum(values) / len(values)


def calc_std(values):
    """回傳樣本標準差。"""
    mean = calc_mean(values)
    variance = sum((v - mean) ** 2 for v in values) / (len(values) - 1)
    return math.sqrt(variance)


def calc_median(values):
    """回傳數字串列的中位數。"""
    s = sorted(values)
    n = len(s)
    mid = n // 2
    if n % 2 == 0:
        return (s[mid - 1] + s[mid]) / 2
    else:
        return s[mid]


def print_stats(label, values):
    """印出描述性統計資料的摘要。"""
    print(f"\n  Statistics for {label}:")
    print(f"    Count:    {len(values)}")
    print(f"    Mean:     {calc_mean(values):.4f}")
    print(f"    Median:   {calc_median(values):.4f}")
    print(f"    Std Dev:  {calc_std(values):.4f}")
    print(f"    Min:      {min(values):.4f}")
    print(f"    Max:      {max(values):.4f}")


# ----- 線性迴歸（最小平方法）-----

def linear_fit(x, y):
    """使用最小平方法擬合直線 y = mx + b。

    實作封閉式解：
        m = (n * sum(x*y) - sum(x) * sum(y)) / (n * sum(x^2) - (sum(x))^2)
        b = mean(y) - m * mean(x)

    Parameters
    ----------
    x : list of float
    y : list of float

    Returns
    -------
    m : float   斜率
    b : float   截距
    r_squared : float   決定係數 (R^2)
    """
    n = len(x)
    sum_x   = sum(x)
    sum_y   = sum(y)
    sum_xy  = sum(xi * yi for xi, yi in zip(x, y))
    sum_x2  = sum(xi ** 2 for xi in x)
    sum_y2  = sum(yi ** 2 for yi in y)

    # 斜率
    denom = n * sum_x2 - sum_x ** 2
    m = (n * sum_xy - sum_x * sum_y) / denom

    # 截距
    b = (sum_y - m * sum_x) / n

    # R 平方
    ss_res = sum((yi - (m * xi + b)) ** 2 for xi, yi in zip(x, y))
    ss_tot = sum((yi - calc_mean(y)) ** 2 for yi in y)
    r_squared = 1 - ss_res / ss_tot if ss_tot != 0 else 0

    return m, b, r_squared


# ----- 繪圖 -----

def plot_results(x, y, m, b, r_squared, filename="numerical_analysis.png"):
    """繪製資料點和擬合線，並儲存到檔案。"""
    # 計算擬合線的值
    y_fit = [m * xi + b for xi in x]

    fig, ax = plt.subplots(figsize=(8, 5))

    # 原始資料的散布圖
    ax.scatter(x, y, color="steelblue", alpha=0.6, s=30, label="Data points")

    # 擬合線
    ax.plot(x, y_fit, color="tomato", linewidth=2,
            label=f"Fit: y = {m:.2f}x + {b:.2f} (R² = {r_squared:.4f})")

    ax.set_title("Numerical Analysis: Linear Fit", fontsize=14, fontweight="bold")
    ax.set_xlabel("x")
    ax.set_ylabel("y")
    ax.legend()
    ax.grid(True, alpha=0.3)

    fig.tight_layout()
    fig.savefig(filename, dpi=150)
    print(f"\n  Plot saved to: {filename}")
    plt.close(fig)

def main():
    """執行完整的分析流程。"""
    print("=" * 50)
    print("  Simple Numerical Analysis")
    print("=" * 50)

    # 步驟 1：產生資料
    x, y = generate_data(n=50, slope=2.5, intercept=10, noise_level=5.0)

專案 9：命令列工具

目標

建構一個 CLI 工具，計算一個或多個文字檔中的字數、行數和字元數 — 類似 Unix 的 wc 命令。使用 argparse 進行適當的引數解析，含說明文字、旗標和多檔案支援。

完整程式碼

# ============================================================
# wordcount.py — 文字檔的 CLI 字數/行數/字元數計數器
# ============================================================
# 用法：
#   python wordcount.py file1.txt file2.txt
#   python wordcount.py *.txt --sort words --top 5
#   python wordcount.py myfile.txt --no-total

import argparse
import sys
from pathlib import Path


def count_file(filepath):
    """計算單一檔案中的行數、字數和字元數。

    Parameters
    ----------
    filepath : str or Path
        文字檔的路徑。

    Returns
    -------
    dict with keys: 'lines', 'words', 'chars', 'file'
    如果檔案無法讀取則回傳 None。
    """
    try:
        with open(filepath, "r", encoding="utf-8") as f:
            content = f.read()
    except FileNotFoundError:
        print(f"  Error: File not found: {filepath}", file=sys.stderr)
        return None
    except PermissionError:
        print(f"  Error: Permission denied: {filepath}", file=sys.stderr)
        return None
    except UnicodeDecodeError:
        print(f"  Error: Cannot read (not a text file?): {filepath}", file=sys.stderr)
        return None

    lines = content.count("\n")
    words = len(content.split())
    chars = len(content)

    return {
        "lines": lines,
        "words": words,
        "chars": chars,
        "file": str(filepath),
    }


def print_results(results, show_total=True, sort_by=None, top_n=None):
    """印出格式化的字數計數結果表格。

    Parameters
    ----------
    results : list of dict
        每個字典來自 count_file()。
    show_total : bool
        是否在底部印出總計列。
    sort_by : str or None
        按 'lines'、'words' 或 'chars' 排序結果（降序）。
    top_n : int or None
        僅顯示前 N 個檔案（排序後）。
    """
    if not results:
        print("No results to display.")
        return

    # 可選排序
    if sort_by and sort_by in ("lines", "words", "chars"):
        results = sorted(results, key=lambda r: r[sort_by], reverse=True)

    # 可選的前 N 篩選
    if top_n is not None and top_n > 0:
        results = results[:top_n]

    # 印出表頭
    print(f"\n{'Lines':>10} {'Words':>10} {'Chars':>10}  {'File'}")
    print("-" * 50)

    # 印出每個檔案的計數
    total_lines = 0
    total_words = 0
    total_chars = 0

    for r in results:
        print(f"{r['lines']:>10,} {r['words']:>10,} {r['chars']:>10,}  {r['file']}")
        total_lines += r["lines"]
        total_words += r["words"]
        total_chars += r["chars"]

    # 印出總計（僅在超過一個檔案且 show_total 為 True 時）
    if show_total and len(results) > 1:
        print("-" * 50)
        print(f"{total_lines:>10,} {total_words:>10,} {total_chars:>10,}  TOTAL")

    print()


def build_parser():
    """建立並回傳引數解析器。"""
    parser = argparse.ArgumentParser(
        description="Count lines, words, and characters in text files.",
        epilog="Example: python wordcount.py report.txt notes.txt --sort words",
    )

    # 位置引數：一個或多個檔案路徑
    parser.add_argument(
        "files",
        nargs="+",
        help="One or more text files to analyze.",
    )

    # 可選旗標
    parser.add_argument(
        "--sort",
        choices=["lines", "words", "chars"],
        default=None,
        help="Sort results by the specified metric (descending).",
    )

    parser.add_argument(
        "--top",
        type=int,
        default=None,
        metavar="N",
        help="Show only the top N files (requires --sort).",
    )

    parser.add_argument(
        "--no-total",
        action="store_true",
        help="Do not show the total row.",
    )

    return parser


def main():
    """解析引數並執行字數計數器。"""
    parser = build_parser()
    args = parser.parse_args()

    # 計算每個檔案
    results = []
    for filepath in args.files:
        result = count_file(filepath)
        if result is not None:
            results.append(result)

    if not results:
        print("No valid files to analyze.", file=sys.stderr)
        sys.exit(1)

    # 顯示結果
    print_results(
        results,
        show_total=not args.no_total,
        sort_by=args.sort,
        top_n=args.top,
    )


if __name__ == "__main__":
    main()
