輸出、註解、變數與數字

這是第一個「真正的 Python」頁面。學完這一頁後，你將能夠在螢幕上顯示輸出、在程式碼中留下註解、將值儲存在變數中，以及進行算術運算。這四項技能構成了你將來撰寫的每一個 Python 程式的基礎。

為什麼這很重要：每一種程式語言都需要一種方式來顯示結果、記住值，以及進行數值運算。掌握這些基礎知識後，接下來的一切—字串、串列、函式—都會感覺像是自然的延伸。

1. print() 函式

print() 是 Python 的內建函式，用來將輸出顯示在螢幕上（「主控台」或「終端機」）。它是你擁有的最實用的除錯和學習工具。當你不確定某個值是什麼時，直接用 print() 來查看它。

運作方式：你透過寫 print 這個字後面加上括號來呼叫 print()。在括號裡放入你想顯示的內容。Python 會將其轉換為文字並寫入主控台，然後加上一個換行字元（這樣下一個輸出就會出現在新的一行）。

1.1 基本輸出

最簡單的用法：將一個字串（引號內的文字）傳給 print()。

# 範例 1 — 印出一個簡單的字串
print("Hello, world!")
# 輸出：Hello, world!

# 範例 2 — 單引號也可以
print('Python is fun!')
# 輸出：Python is fun!

# 範例 3 — 直接印出一個數字
print(42)
# 輸出：42

1.2 印出多個值

你可以用逗號分隔的方式傳遞多個值給 print()。Python 會將它們全部印在同一行，預設用空格分隔。

# 範例 4 — 多個值
print("The answer is", 42)
# 輸出：The answer is 42

# 範例 5 — 自由混合不同型別
print("Pi is approximately", 3.14159, "and e is approximately", 2.71828)
# 輸出：Pi is approximately 3.14159 and e is approximately 2.71828

1.3 sep 參數

sep（分隔符）參數控制每個值之間放什麼。預設是一個空格（" "）。

# 範例 6 — 自訂分隔符
print("2026", "03", "31", sep="-")
# 輸出：2026-03-31

# 範例 7 — 無分隔符
print("P", "y", "t", "h", "o", "n", sep="")
# 輸出：Python

# 範例 8 — Tab 分隔符
print("Name", "Age", "City", sep="\t")
# 輸出：Name    Age    City

1.4 end 參數

預設情況下 print() 以換行字元（\n）結尾，因此下一個 print() 會出現在下一行。end 參數讓你改變這個行為。

# 範例 9 — 在同一行印出
print("Loading", end="")
print("...", end="")
print(" Done!")
# 輸出：Loading... Done!

# 範例 10 — 以自訂字串結尾
print("Item 1", end=" | ")
print("Item 2", end=" | ")
print("Item 3")
# 輸出：Item 1 | Item 2 | Item 3

1.5 印出變數

你幾乎總是會印出變數的內容，而不是寫死的文字。只需將變數名稱放在 print() 裡面—不需要加引號。

# 範例 11 — 印出變數
name = "Alice"
age = 30
print(name)          # 輸出：Alice
print(age)           # 輸出：30
print(name, "is", age, "years old.")
# 輸出：Alice is 30 years old.
常見錯誤 — 忘記括號：
在 Python 2 中，print 是一個陳述式，你可以寫 print "hello"。在 Python 3 中，print() 是一個函式，括號是必須的。如果你看到舊的教學或 Stack Overflow 的答案使用 print "hello"，請知道這在 Python 3 中會產生 SyntaxError。 
# Python 3 中的錯誤寫法：
print "hello"   # SyntaxError: Missing parentheses in call to 'print'

# 正確寫法：
print("hello")
MATLAB 注意事項： print() 類似於 MATLAB 的 disp()（用於簡單輸出）和 fprintf()（用於格式化輸出）。主要差異是：Python 的 print() 會自動在結尾加上換行（像 disp()），而 MATLAB 的 fprintf() 則不會，除非你自己加上 \n。在 Python 中，如果你想要格式化輸出，請查看 f-string（在字串頁面介紹），它取代了 fprintf 風格的格式化。 
% MATLAB
disp('Hello, world!')
fprintf('The answer is %d\n', 42)

# Python 等效寫法
print('Hello, world!')
print(f'The answer is {42}')

2. 註解

註解是你在程式碼中為自己（或其他閱讀程式碼的人）留下的備註。Python 在執行程式時會完全忽略註解—它們存在的目的僅僅是讓程式碼更容易理解。

為什麼這很重要：程式碼被閱讀的次數遠比被撰寫的次數多。一個放置得當的註解，可以在你幾週或幾個月後重新檢視程式碼時，省下你（或同事）數小時的困惑。

2.1 使用 # 的單行註解

一行中 # 之後的所有內容都是註解。Python 會忽略它。

# 這是一個註解 — Python 會跳過整行
print("Hello")  # 這個註解在程式碼之後 — Python 執行 print，忽略其餘部分

# 你可以用註解來暫時停用程式碼：
# print("This line will NOT run")

2.2 使用三引號的多行註解

Python 沒有專門的多行註解語法（不像 C 語言的 /* ... */）。然而，你可以使用三引號字串（"""...""" 或 '''...'''）作為替代。如果三引號字串沒有被指定給變數，Python 會評估它然後丟棄—實際上使其成為註解。

"""
This is a multi-line comment (technically a string literal).
Python evaluates it but discards the result,
so it behaves like a comment.
"""

'''
You can also use single triple-quotes.
This is useful for longer explanations.
'''
文件字串（Docstrings）：當三引號字串出現在函式、類別或模組內部的第一個陳述式時，它會成為文件字串—一種特殊的文件，Python 會儲存它並透過 help() 函式讓你可以存取。你將在函式頁面詳細學習文件字串。 
def greet(name):
    """為指定的名字回傳一個問候字串。"""
    return f"Hello, {name}!"

help(greet)
# 顯示：為指定的名字回傳一個問候字串。

2.3 何時使用註解

  • 解釋「為什麼」，而非「做什麼」。好的例子：# 轉換為公尺，因為感測器回傳的是毫米。不好的例子：# 將 x 除以 1000（程式碼已經說明了這一點）。
  • 標記待辦事項： # TODO: 處理檔案為空的情況。
  • 在除錯時暫時停用程式碼（稱為「註解掉」）。
  • 長腳本中的區段標題：# --- 資料載入 ---。
常見錯誤 — 過度註解：不要在每一行都加上註解。如果你的變數名稱是 total_price，你不需要加上 # 總價 的註解。使用清楚的變數名稱，把註解留給不明顯的邏輯。
MATLAB 注意事項： MATLAB 使用 % 作為單行註解，使用 %{ ... %} 作為區塊註解。在 Python 中，使用 # 作為單行，使用 """...""" 作為多行區塊。對應關係很直觀： 
% MATLAB 單行註解
%{
  MATLAB 區塊註解
%}

# Python 單行註解
"""
Python 多行註解
（如果在函式/類別內則為文件字串）
"""

3. 變數

3.1 什麼是變數？

變數是一個名稱，它指向儲存在電腦記憶體中的一個值。把它想像成一個有標籤的盒子：標籤是變數名稱，盒子裡的內容就是值。你可以把東西放進盒子、查看裡面有什麼、用其他東西替換，或在計算中使用它。

白話解釋：變數只是一種為一筆資料命名的方式，讓你以後可以使用它。你不用到處寫 3.14159，只要寫一次 pi = 3.14159，然後在需要的地方使用 pi 這個名稱即可。

3.2 使用 = 賦值

你使用 = 運算子來建立（或更新）一個變數。名稱放在左邊；值放在右邊。

# 範例 12 — 建立變數
message = "Hello, Python!"
count = 5
temperature = 98.6

print(message)        # 輸出：Hello, Python!
print(count)          # 輸出：5
print(temperature)    # 輸出：98.6

你可以隨時重新賦值一個變數—舊的值會直接被替換。

# 範例 13 — 重新賦值
x = 10
print(x)   # 輸出：10

x = 20
print(x)   # 輸出：20

x = x + 5  # 使用 x 的當前值（20）來計算新值（25）
print(x)   # 輸出：25

3.3 命名規則與慣例

Python 有嚴格的規則（違反會產生錯誤）以及社群的慣例（違反會讓程式碼更難閱讀）。

規則 / 慣例範例說明
必須以字母或底線開頭name, _private不能以數字開頭
可以包含字母、數字、底線score_1, total2不能有空格、連字號或特殊字元
區分大小寫Name ≠ name ≠ NAME這是三個不同的變數
慣例：使用 snake_casetotal_price, user_name小寫字母以底線分隔
慣例：常數使用大寫MAX_RETRIES = 5表示「不要改變這個值」
不要使用保留字避免使用 print、list、type這些在 Python 中有特殊意義
 
# 好的變數名稱
student_name = "Bob"
max_score = 100
is_active = True
_internal_counter = 0

# 合法但不建議（不是 snake_case）
studentName = "Bob"   # 這是 camelCase — 在 JavaScript 中常見，Python 中不建議
StudentName = "Bob"   # 這是 PascalCase — 在 Python 中保留給類別名稱

3.4 動態型別

在 Python 中，你不需要宣告變數的型別。Python 會根據你賦予的值自動判斷。這稱為動態型別。

# 範例 14 — 動態型別
x = 10          # x 是整數
print(type(x))  # 輸出：<class 'int'>

x = "hello"     # 現在 x 是字串 — 不會出錯！
print(type(x))  # 輸出：<class 'str'>

x = 3.14        # 現在 x 是浮點數
print(type(x))  # 輸出：<class 'float'>
type() 函式：你可以隨時呼叫 type(variable_name) 來檢查變數目前的型別。這對除錯非常有用。你在學習過程中會經常使用它。

3.5 多重賦值

Python 允許你在一行中指定多個變數。這是一個方便的簡寫方式。

# 範例 15 — 單行多重賦值
x, y, z = 1, 2, 3
print(x, y, z)   # 輸出：1 2 3

# 將同一個值指定給多個變數
a = b = c = 0
print(a, b, c)   # 輸出：0 0 0

# 不需要臨時變數就能交換兩個變數
x, y = 10, 20
print(x, y)       # 輸出：10 20
x, y = y, x       # 交換！
print(x, y)       # 輸出：20 10
為什麼這很重要：變數交換技巧（x, y = y, x）是 Python 讓事情變得優美簡潔的一個例子。在大多數其他語言（包括 MATLAB）中，你需要一個臨時變數來交換兩個值。在 Python 中只需一行。
常見錯誤 — 使用保留字： Python 有大約 35 個關鍵字（如 if、for、while、True、False、None、class、return），你不能用它們作為變數名稱。否則會得到 SyntaxError。 
# 錯誤 — 'class' 是保留關鍵字
class = "Biology 101"   # SyntaxError: invalid syntax

# 修正 — 使用稍微不同的名稱
course_class = "Biology 101"
此外，避免覆蓋內建名稱如 print、list、type、sum、max 等。這不會立即產生錯誤，但會使這些函式在後續程式碼中無法使用。 
# 不好 — 覆蓋了內建的 list() 函式
list = [1, 2, 3]
# 現在 list() 不再如預期般運作！

# 修正 — 使用描述性名稱
my_list = [1, 2, 3]
常見錯誤 — 以數字開頭： 
# 錯誤
2nd_place = "Silver"   # SyntaxError: invalid syntax

# 修正
second_place = "Silver"
place_2 = "Silver"
MATLAB 注意事項： MATLAB 和 Python 中的變數建立方式非常相似—兩者都使用 =，且都不需要型別宣告。主要差異：
  • MATLAB 慣例使用 camelCase；Python 慣例使用 snake_case。
  • MATLAB 在行尾使用 ; 來抑制輸出；Python 不會自動印出賦值結果（沒有東西需要抑制）。
  • Python 變數區分大小寫。MATLAB 變數也區分大小寫，所以這一點相同。
  • MATLAB 的 who/whos 指令在 Python 中沒有直接對應，但你可以使用 dir() 或 type() 來檢視變數。
% MATLAB
studentName = 'Alice';   % camelCase，分號抑制輸出
x = 10;

# Python
student_name = "Alice"   # snake_case，不需要分號
x = 10

練習 3 — 練習使用變數

  1. 建立你的名字、姓氏和年齡的變數。
  2. 用一個句子印出它們，如：Alice Smith is 30 years old.
  3. 使用單行交換技巧交換你的名字和姓氏，然後印出結果。
  4. 使用 type() 印出每個變數的型別。
顯示解答 
# 步驟 1：建立變數
first_name = "Alice"
last_name = "Smith"
age = 30

# 步驟 2：印出一個句子
print(first_name, last_name, "is", age, "years old.")
# 輸出：Alice Smith is 30 years old.

# 步驟 3：交換名字和姓氏

# 輸出：After swap: Smith Alice

# 步驟 4：檢查型別
print(type(first_name))   # <class 'str'>
print(type(last_name))    # <class 'str'>
print(type(age))          # <class 'int'>

4. 數字：整數與浮點數

Python 有兩種你每天都會用到的數值型別：

  • int（整數）—沒有小數點的整數：0、42、-7、1000000
  • float（浮點數）—有小數點的數字：3.14、-0.001、2.0
# 範例 16 — int 與 float
a = 10       # int
b = 10.0     # float（.0 使其成為 float）
c = -3       # int
d = 0.5      # float

print(type(a))  # <class 'int'>
print(type(b))  # <class 'float'>
關鍵差異： 10 是 int，而 10.0 是 float。它們的值相等（10 == 10.0 是 True），但它們是不同的型別。這很少造成問題，但了解這一點很有幫助。

4.1 算術運算子

Python 支援你預期的所有算術運算，加上一些額外的。

運算子名稱範例結果
+加法10 + 313
-減法10 - 37
*乘法10 * 330
/除法（真除法）10 / 33.3333...
//整數除法10 // 33
%取餘數10 % 31
**次方10 ** 31000
 
# 範例 17 — 所有算術運算子
a = 17
b = 5

print("a + b  =", a + b)     # 22
print("a - b  =", a - b)     # 12
print("a * b  =", a * b)     # 85
print("a / b  =", a / b)     # 3.4
print("a // b =", a // b)    # 3    （整數除法 — 向下取整）
print("a % b  =", a % b)     # 2    （17 / 5 的餘數）
print("a ** b =", a ** b)    # 1419857  （17 的 5 次方）

4.2 整數除法 vs 浮點除法

這是最重要的概念之一：

  • /（單斜線）總是回傳浮點數，即使結果是整數。
  • //（雙斜線）回傳結果向下取整到最接近的整數（整數除法）。
# 範例 18 — 除法比較
print(10 / 2)     # 5.0   （浮點數，即使 10 可以被 2 整除）
print(10 // 2)    # 5     （整數）

print(7 / 2)      # 3.5   （浮點數）
print(7 // 2)     # 3     （整數 — 向下取整，不是四捨五入）

print(-7 / 2)     # -3.5  （浮點數）
print(-7 // 2)    # -4    （向負無窮取整，不是向零取整！）
常見錯誤 — 除法的意外行為：在 Python 3 中，/ 總是回傳浮點數。這讓來自 C、Java 或 Python 2 的人感到驚訝，因為在那些語言中，兩個整數相除會得到整數。 
print(10 / 5)    # 2.0，不是 2
print(type(10 / 5))  # <class 'float'>
如果你想要整數結果，使用 //。另外注意：// 是向負無窮取整，不是向零取整。所以 -7 // 2 是 -4，不是 -3。

4.3 運算順序（優先級）

Python 遵循標準的數學運算順序，通常記為 PEMDAS：

  1. Parentheses（括號）()
  2. Exponentiation（次方）**
  3. Multiplication（乘法）*、Division（除法）/、整數除法 //、取餘數 %（由左至右）
  4. Addition（加法）+、Subtraction（減法）-（由左至右）
# 範例 19 — 運算順序
print(2 + 3 * 4)        # 14  （先乘法：3*4=12，然後 2+12=14）
print((2 + 3) * 4)      # 20  （先括號：2+3=5，然後 5*4=20）
print(2 ** 3 ** 2)       # 512 （次方由右至左：3**2=9，然後 2**9=512）
print(10 - 2 * 3 + 1)   # 5   （2*3=6，然後 10-6+1=5）
提示：有疑問時就用括號。它們讓你的意圖一目瞭然，並防止錯誤。(a + b) * c 一看就懂；a + b * c 則需要你記住優先級規則。

4.4 常用內建數學函式

Python 提供了幾個不需要 import 的數學函式。

# 範例 20 — 內建數學函式
print(abs(-42))          # 42    （絕對值）
print(abs(3.7))          # 3.7

print(round(3.14159))    # 3     （四捨五入到最接近的整數）
print(round(3.14159, 2)) # 3.14  （四捨五入到小數點後 2 位）
print(round(2.5))        # 2     （銀行家取整 — 取到最接近的偶數）
print(round(3.5))        # 4

print(min(4, 7, 2, 9))   # 2     （最小值）
print(max(4, 7, 2, 9))   # 9     （最大值）

print(pow(2, 10))        # 1024  （等同 2 ** 10）
常見錯誤 — 銀行家取整： Python 的 round() 使用「四捨六入五取偶」（銀行家取整）。這表示 round(0.5) 是 0，round(1.5) 是 2，round(2.5) 是 2，round(3.5) 是 4。這在統計上是無偏的，但會讓大多數初學者感到驚訝。如果你需要傳統的「四捨五入」行為，你需要使用不同的方法（例如 math.floor(x + 0.5)）。

4.5 math 模組

對於更進階的數學運算，Python 提供了 math 模組。你必須在使用前匯入它。

# 範例 21 — math 模組
import math

# 平方根
print(math.sqrt(144))     # 12.0
print(math.sqrt(2))       # 1.4142135623730951

# 常數
print(math.pi)            # 3.141592653589793
print(math.e)             # 2.718281828459045

# 天花板函式和地板函式
print(math.ceil(4.2))     # 5   （向上取整到最接近的整數）
print(math.floor(4.8))    # 4   （向下取整到最接近的整數）
print(math.ceil(-4.2))    # -4  （向正無窮方向）
print(math.floor(-4.8))   # -5  （向負無窮方向）

# 對數
print(math.log(100, 10))  # 2.0  （以 10 為底的 100 的對數）
print(math.log(math.e))   # 1.0  （e 的自然對數）

# 三角函式（引數以弧度為單位）
print(math.sin(math.pi / 2))  # 1.0
print(math.cos(0))            # 1.0

# 角度與弧度互換
print(math.radians(180))      # 3.141592653589793
print(math.degrees(math.pi))  # 180.0
MATLAB 注意事項： MATLAB 預設將所有數字視為 double（64 位元浮點數）。Python 區分 int 和 float。這意味著：
  • 在 MATLAB 中，x = 10 建立的是 double。在 Python 中，x = 10 建立的是 int，而 x = 10.0 建立的是 float。
  • MATLAB 的數學函式（sqrt、sin、abs 等）可以直接使用。在 Python 中，大部分需要先 import math（abs()、round()、min()、max() 是內建的，但 sqrt()、sin()、cos() 在 math 模組中）。
  • MATLAB 使用 ^ 做次方運算；Python 使用 **。
  • MATLAB 的 mod(a, b) 等同 Python 的 a % b。
% MATLAB
x = sqrt(144);        % 12（double）
y = 10 ^ 3;           % 1000
r = mod(17, 5);       % 2

# Python
import math
x = math.sqrt(144)    # 12.0（float）
y = 10 ** 3           # 1000（int）
r = 17 % 5            # 2（int）

4.6 型別檢查與轉換

你可以使用 type() 檢查值的型別，並使用 int()、float() 和 str() 在型別之間轉換。

# 範例 22 — 型別檢查
x = 42
print(type(x))         # <class 'int'>
print(isinstance(x, int))   # True
print(isinstance(x, float)) # False
# 範例 23 — 型別轉換
# float 轉 int（截斷 — 不會四捨五入）
print(int(3.9))        # 3
print(int(-3.9))       # -3

# int 轉 float
print(float(42))       # 42.0

# 字串轉數字
print(int("100"))      # 100
print(float("3.14"))   # 3.14

# 數字轉字串
print(str(42))         # "42"
print(str(3.14))       # "3.14"
常見錯誤 — int() 是截斷，不是四捨五入： 
print(int(9.99))    # 9，不是 10
print(int(-2.7))    # -2，不是 -3

# 如果你要四捨五入，先使用 round()：
print(round(9.99))  # 10
print(int(round(9.99)))  # 10

4.7 複合賦值運算子

Python 提供了用算術運算更新變數的簡寫方式：

# 範例 24 — 複合賦值
x = 10

x += 5    # 等同：x = x + 5   → x 現在是 15
x -= 3    # 等同：x = x - 3   → x 現在是 12
x *= 2    # 等同：x = x * 2   → x 現在是 24
x /= 4    # 等同：x = x / 4   → x 現在是 6.0（float！）
x //= 2   # 等同：x = x // 2  → x 現在是 3.0
x **= 3   # 等同：x = x ** 3  → x 現在是 27.0
x %= 5    # 等同：x = x % 5   → x 現在是 2.0

print(x)  # 2.0
注意：一旦使用了 /=，變數就會變成 float，並在後續所有操作中保持 float，即使是 //= 也一樣。如果你從 x = 10（int）開始，然後執行 x /= 2，x 現在是 5.0（float），不是 5（int）。

4.8 大數字與底線

Python 可以處理任意大的整數（不會溢位！），並允許你在數字中使用底線作為視覺分隔符。

# 範例 25 — 大數字與底線
big = 1_000_000_000     # 比 1000000000 更容易閱讀
print(big)               # 1000000000

# Python 的整數大小沒有限制
huge = 2 ** 100
print(huge)              # 1267650600228229401496703205376

# 底線在浮點數中也可以使用
speed_of_light = 299_792_458.0   # 公尺/秒
print(speed_of_light)            # 299792458.0

練習 4 — 算術練習

一個長方形的寬度為 7.5，高度為 12.3。寫一個 Python 腳本：

  1. 將寬度和高度儲存在變數中。
  2. 計算並印出面積（寬 × 高）。
  3. 計算並印出周長（2 ×（寬 + 高））。
  4. 使用畢氏定理計算並印出對角線（使用 math.sqrt()）。
  5. 將每個結果四捨五入到小數點後 2 位。
顯示解答 
import math

# 步驟 1：儲存尺寸
width = 7.5
height = 12.3

# 步驟 2：面積
area = width * height
print("Area:", round(area, 2))         # 92.25

# 步驟 3：周長
perimeter = 2 * (width + height)
print("Perimeter:", round(perimeter, 2))  # 39.6

# 步驟 4：對角線（畢氏定理）
diagonal = math.sqrt(width ** 2 + height ** 2)
print("Diagonal:", round(diagonal, 2))   # 14.41

練習 5 — 溫度轉換器

寫一個腳本，使用以下公式將溫度從華氏轉換為攝氏：

celsius = (fahrenheit - 32) * 5 / 9

  1. 將 98.6（正常體溫）儲存在名為 fahrenheit 的變數中。
  2. 計算攝氏等效值並儲存在名為 celsius 的變數中。
  3. 印出結果，四捨五入到小數點後 1 位。
  4. 印出兩個變數的型別。
顯示解答 
# 溫度轉換
fahrenheit = 98.6
celsius = (fahrenheit - 32) * 5 / 9

print("Fahrenheit:", fahrenheit)
print("Celsius:", round(celsius, 1))      # 37.0

print("Type of fahrenheit:", type(fahrenheit))  # <class 'float'>
print("Type of celsius:", type(celsius))        # <class 'float'>

練習 6 — 整數除法與取餘數

你有 157 分鐘的影片要看。寫一個腳本：

  1. 使用整數除法（//）來計算有多少完整的小時。
  2. 使用取餘數（%）來計算剩餘的分鐘數。
  3. 印出結果如：157 minutes = 2 hours and 37 minutes

total_minutes = 157

hours = total_minutes // 60
remaining = total_minutes % 60

print(total_minutes, "minutes =", hours, "hours and", remaining, "minutes")
