最大的運算子差異

MATLAB：* = 矩陣乘法，.* = 元素逐一運算。
NumPy：@ = 矩陣乘法，* = 元素逐一運算。
這是將 MATLAB 翻譯為 Python 時最常見的隱性錯誤來源。

pyplot 介面刻意模仿 MATLAB 的繪圖指令。大多數函式名稱 相同或幾乎相同。最大的差異：加上 plt. 前綴、 使用關鍵字引數取代名稱-值配對、以及總是要呼叫 plt.show()。

匯入速查表

大多數 MATLAB 功能是內建的。在 Python 中，你需要 import 程式庫。以下是進行類似 MATLAB 工作時的標準匯入： 

# 標準科學 Python 匯入
import numpy as np              # 陣列、數學（類似 MATLAB 核心功能）
import matplotlib.pyplot as plt # 繪圖（類似 MATLAB figure/plot）
import pandas as pd             # 表格/DataFrame（類似 MATLAB 表格）
from scipy import linalg        # 進階線性代數
from scipy import optimize      # 最佳化（類似 fmincon）
from scipy import integrate     # 積分（類似 integral）
from scipy import signal        # 信號處理（類似 DSP 工具箱）
from scipy import stats         # 統計（類似 Statistics 工具箱）
import os                       # 檔案系統操作
import math                     # 基本數學（sqrt、pi 等）
提示：每個科學腳本都以這些開頭

在每個腳本頂部放上 import numpy as np 和 import matplotlib.pyplot as plt。這兩個匯入 提供了 MATLAB 90% 的核心功能。如果你處理表格/CSV 資料， 再加上 import pandas as pd。
