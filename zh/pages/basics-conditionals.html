第 C5 節 — 條件判斷 (if / elif / else)

每個有用的程式都需要做決定。是否應該顯示錯誤訊息？ 使用者年齡是否足夠？感測器讀數是否超過閾值？ 在 Python 中，你使用條件語句 — if、elif 和 else 來做決定。 本頁涵蓋你需要了解的所有 Python 分支邏輯知識。

MATLAB 使用者注意： 你已經知道 if ... elseif ... else ... end。 Python 的版本在邏輯上幾乎相同，但語法有三個重要差異： (1) Python 使用 elif 而非 elseif，(2) 每個條件行以 冒號結尾，(3) 沒有 end 關鍵字 — Python 僅使用縮排來定義區塊。

1. if 語句

最簡單的條件是純粹的 if。它僅在條件為 True 時執行一段程式碼區塊。

語法

if condition:
    # 此區塊僅在 condition 為 True 時執行
    do_something()

關鍵規則：

  • 該行必須以冒號（:）結尾。
  • 下方的縮排區塊是主體。Python 使用縮排（通常 4 個空格）來定義哪些行屬於 if。
  • 條件是任何求值為 True 或 False 的表達式。

範例 1 — 基本 if

temperature = 35

if temperature > 30:
    print("It's hot outside!")

# 輸出：It's hot outside!

因為 temperature 是 35，條件 35 > 30 為 True，所以縮排的 print() 行會被執行。

範例 2 — 條件為 False 時什麼都不會發生

temperature = 20

if temperature > 30:
    print("It's hot outside!")

print("Done checking.")
# 輸出：Done checking.

條件為 False，所以縮排的行被完全跳過。 程式繼續執行下一行未縮排的程式碼。

縮排不是可選的。 在大多數語言中， 縮排只是風格選擇。在 Python 中，它是語法的一部分。 如果你縮排不正確，Python 會引發 IndentationError。始終使用每層 4 個空格 （你的編輯器可以自動完成此操作）。

2. if / else

通常你想在條件為真時做一件事，在為假時做 另一件事。這就是 else 的用途。

語法

if condition:
    # 條件為 True 時執行
else:
    # 條件為 False 時執行

範例 3 — 及格或不及格

score = 72

if score >= 60:
    print("You passed!")
else:
    print("You failed. Try again.")

# 輸出：You passed!

範例 4 — 偶數或奇數

number = 7

if number % 2 == 0:
    print(f"{number} is even")
else:
    print(f"{number} is odd")

# 輸出：7 is odd

取模運算子 % 給出除法後的餘數。 如果 number % 2 等於 0，該數字是偶數；否則是奇數。

3. elif — 多重條件

當你有兩個以上的分支時，使用 elif（「else if」的縮寫）。 Python 從上到下檢查每個條件，並執行 第一個條件為 True 的區塊。如果 都不符合，else 區塊會執行（如果有的話）。

語法

if condition_1:
    # 如果 condition_1 為 True 則執行
elif condition_2:
    # 如果 condition_1 為 False 且 condition_2 為 True 則執行
elif condition_3:
    # 如果以上兩者都為 False 且 condition_3 為 True 則執行
else:
    # 如果以上所有條件都為 False 則執行

你可以有任意多個 elif 分支。 末尾的 else 是可選的，但建議作為兜底使用。

範例 5 — 字母等級

score = 85

if score >= 90:
    grade = "A"
elif score >= 80:
    grade = "B"
elif score >= 70:
    grade = "C"
elif score >= 60:
    grade = "D"
else:
    grade = "F"

print(f"Score: {score}, Grade: {grade}")
# 輸出：Score: 85, Grade: B

Python 依序測試每個條件。85 >= 90 為 False，所以繼續。85 >= 80 為 True，所以 grade 被設為 "B"， 所有剩餘分支被跳過。

MATLAB 比較： 
% MATLAB
if score >= 90
    grade = 'A';
elseif score >= 80
    grade = 'B';
elseif score >= 70
    grade = 'C';
elseif score >= 60
    grade = 'D';
else
    grade = 'F';
end

注意 MATLAB 使用 elseif（一個字）且需要 end。Python 使用 elif 並用縮排取代 end。邏輯上完全相同。

4. 巢狀 if 語句

你可以將一個 if 語句放在另一個 if 語句內部。這稱為巢狀。 內部的 if 只在外部條件已經為 True 時才會執行。

範例 6 — 巢狀檢查

age = 25
has_ticket = True

if age >= 18:
    if has_ticket:
        print("Welcome to the concert!")
    else:
        print("You need a ticket.")
else:
    print("Sorry, you must be 18 or older.")

# 輸出：Welcome to the concert!

每層巢狀增加 4 個空格的縮排。雖然巢狀可以運作， 但深層巢狀的程式碼（3 層以上）會變得難以閱讀。 在實務上，你通常可以使用 and 組合來簡化巢狀條件（見下文）。

範例 7 — 使用 and 的扁平版本

age = 25
has_ticket = True

if age >= 18 and has_ticket:
    print("Welcome to the concert!")
elif age < 18:
    print("Sorry, you must be 18 or older.")
else:
    print("You need a ticket.")

這樣更簡潔、更容易閱讀。盡可能選擇扁平化。

5. 使用 and、or、not 組合條件

Python 提供三個邏輯運算子來組合條件：

運算子 含義 何時為 True…
and 兩者都必須為真 A and B → 僅當 A 為 True 且 B 為 True 時為 True
or 至少一個必須為真 A or B → 如果 A 為 True 或 B 為 True（或兩者）則為 True
not 反轉真假值 not A → 如果 A 為 False 則為 True

範例 8 — 使用 and

username = "admin"
password = "secret123"

if username == "admin" and password == "secret123":
    print("Login successful!")
else:
    print("Invalid credentials.")

# 輸出：Login successful!

兩個條件都必須為 True，組合表達式才為 True。

範例 9 — 使用 or

day = "Saturday"

if day == "Saturday" or day == "Sunday":
    print("It's the weekend!")
else:
    print("It's a weekday.")

# 輸出：It's the weekend!

範例 10 — 使用 not

is_raining = False

if not is_raining:
    print("No umbrella needed.")

# 輸出：No umbrella needed.

not 將 False 翻轉為 True， 所以區塊會被執行。

範例 11 — 組合多個運算子

temperature = 22
is_sunny = True

if temperature > 20 and temperature < 35 and is_sunny:
    print("Perfect day for a walk!")

# 輸出：Perfect day for a walk!
運算子優先順序： not 最先求值， 然後是 and，最後是 or。 有疑問時，使用括號讓你的意圖更清楚： if (a or b) and c:
MATLAB 比較： MATLAB 使用 && 作為邏輯 AND，|| 作為邏輯 OR。Python 使用英文單字拼寫： and 和 or。 MATLAB 的 ~（波浪號表示 NOT）在 Python 中變為 not。

6. 比較運算子參考

條件依賴比較運算子。以下是完整列表：

運算子 含義 範例 結果
==等於5 == 5True
!=不等於5 != 3True
>大於5 > 3True
<小於3 < 5True
>=大於或等於5 >= 5True
<=小於或等於3 <= 5True
in成員測試"a" in "cat"True
not in否定成員測試"x" not in "cat"True
is身份（同一物件）x is None取決於情況
鏈式比較： Python 讓你可以自然地鏈式比較。 不必寫 x > 0 and x < 100，你可以寫 0 < x < 100。這就像數學記號一樣， 在 Python 中完全有效。

範例 12 — 鏈式比較

age = 25

if 18 <= age <= 65:
    print("Working age")

# 輸出：Working age

7. 三元表達式（行內 if/else）

有時你想根據條件在單行中賦值。Python 有一個簡潔的語法， 稱為三元表達式（也稱為條件表達式）。

語法

value = a if condition else b

這讀作：「如果條件為真，將 value 設為 a； 否則，設為 b。」

範例 13 — 簡單的三元

age = 20

status = "adult" if age >= 18 else "minor"
print(status)
# 輸出：adult

範例 14 — print 語句中的三元

score = 45

print("Pass" if score >= 50 else "Fail")
# 輸出：Fail

範例 15 — 三元用於選擇值

x = -7

absolute_value = x if x >= 0 else -x
print(absolute_value)
# 輸出：7
何時使用三元： 它們非常適合簡單的單行賦值。 如果邏輯複雜或涉及多個分支， 請使用完整的 if/elif/else 區塊。可讀性永遠勝過巧妙。
MATLAB 注意： MATLAB 沒有內建的三元運算子。 最接近的等價物是單行 if 或使用索引技巧。 Python 的三元語法更簡潔且更易讀。

8. 真值與假值

在 Python 中，你不一定需要明確的比較。每個值都有固有的 真值性。當你寫 if x: 時， Python 會判斷 x 是「真值」還是「假值」。

假值（求值為 False）

值型別
Falsebool
NoneNoneType
0int
0.0float
""（空字串）str
[]（空串列）list
{}（空字典）dict
()（空元組）tuple
set()（空集合）set

其他所有值都是真值。 非零數字、非空字串、非空集合 — 都是真值。

範例 16 — 檢查空串列

shopping_list = []

if shopping_list:
    print("Let's go shopping!")
else:
    print("Nothing to buy.")

# 輸出：Nothing to buy.

空串列是假值，所以 if shopping_list: 求值為 False。這是 Python 中檢查集合是否有項目的慣用方式 — 你不需要寫 if len(shopping_list) > 0:。

範例 17 — 檢查非空字串

user_input = "hello"

if user_input:
    print(f"You typed: {user_input}")
else:
    print("You didn't type anything.")

# 輸出：You typed: hello
為什麼這很重要： 理解真值性是寫出簡潔、 道地 Python 程式碼的關鍵。與其寫冗長的比較如 if name != "": 或 if count != 0:， 有經驗的 Python 程式設計師會寫 if name: 和 if count:。這更短、更簡潔，被視為最佳實踐。

9. 常見錯誤

條件判斷在概念上很簡單，但初學者常常在幾個可預見的 陷阱上犯錯。以下是最常見的錯誤及修正方法。

錯誤 1：使用 = 而非 ==

單等號 = 用於賦值（儲存值）。 雙等號 == 用於比較（檢查是否相等）。 混淆兩者是初學者最常見的錯誤之一。 

# 錯誤 — 這是賦值，不是比較
x = 10
if x = 10:       # SyntaxError!
    print("ten")

# 正確 — 使用 == 進行比較
x = 10
if x == 10:
    print("ten")  # 輸出：ten
好消息： 與 C 或 JavaScript 不同，如果你在 if 中不小心使用了 =，Python 會引發 SyntaxError。錯誤訊息會指出問題所在。

錯誤 2：忘記冒號

# 錯誤 — 結尾缺少冒號
if score > 90
    print("A+")

# 正確
if score > 90:
    print("A+")

每個 if、elif 和 else 行 必須以冒號結尾。忘記它會產生 SyntaxError。

錯誤 3：縮排錯誤

# 錯誤 — 不一致的縮排
if temperature > 30:
    print("Hot")
      print("Drink water")   # IndentationError!

# 正確 — 兩行有相同的縮排
if temperature > 30:
    print("Hot")
    print("Drink water")

同一區塊中的所有行必須有完全相同的縮排。 混用 Tab 和空格是問題的根源 — 將你的編輯器設定為按 Tab 時插入 4 個空格。

錯誤 4：把 elif 寫成 else if

# 錯誤 — Python 不使用 "else if"
if x > 0:
    print("positive")
else if x < 0:       # SyntaxError!
    print("negative")

# 正確 — 使用 elif（一個字）
if x > 0:
    print("positive")
elif x < 0:
    print("negative")

錯誤 5：錯誤地檢查多個值

# 錯誤 — 不會如你所想的那樣運作
color = "red"
if color == "red" or "blue":   # 永遠為 True！
    print("Match")

# 為什麼？Python 將此讀為：(color == "red") or ("blue")
# 字串 "blue" 是真值，所以整個表達式永遠為 True。

# 正確
if color == "red" or color == "blue":
    print("Match")

# 更好 — 使用 'in' 運算子
if color in ("red", "blue"):
    print("Match")
為什麼這很重要： color == "red" or "blue" 這個 bug 在初學者中極為常見。 它在英文中看起來合乎邏輯（「if color equals red or blue」）， 但 Python 的解讀不同。 對於檢查一組值中的成員，始終使用 in。

10. MATLAB vs. Python — 條件判斷並排比較

如果你來自 MATLAB，這個比較將幫助你看到語法如何對應。

功能 MATLAB Python
基本 if if x > 0
  disp('positive');
end		 if x > 0:
    print('positive')
else else
  disp('not positive');
end		 else:
    print('not positive')
Else-if 關鍵字 elseif elif
區塊終止符 end 縮排（沒有 end 關鍵字）
邏輯 AND && and
邏輯 OR || or
邏輯 NOT ~ not

11. 實際範例

範例 18 — 簡單計算機

a = 10
b = 3
operator = "+"

if operator == "+":
    result = a + b
elif operator == "-":
    result = a - b
elif operator == "*":
    result = a * b
elif operator == "/":
    if b != 0:
        result = a / b
    else:
        result = "Error: division by zero"
else:
    result = "Unknown operator"

print(f"{a} {operator} {b} = {result}")
# 輸出：10 + 3 = 13

範例 19 — 閏年檢查器

year = 2024

if year % 400 == 0:
    is_leap = True
elif year % 100 == 0:
    is_leap = False
elif year % 4 == 0:
    is_leap = True
else:
    is_leap = False

print(f"{year} is a leap year: {is_leap}")
# 輸出：2024 is a leap year: True

範例 20 — FizzBuzz（經典程式設計練習）

n = 15

if n % 3 == 0 and n % 5 == 0:
    print("FizzBuzz")
elif n % 3 == 0:
    print("Fizz")
elif n % 5 == 0:
    print("Buzz")
else:
    print(n)

# 輸出：FizzBuzz

注意：順序很重要。組合檢查（n % 3 == 0 and n % 5 == 0）必須放在最前面。如果你先檢查 n % 3 == 0，它會印出「Fizz」並完全跳過 FizzBuzz 的情況。

12. 練習題

練習使之永久。 在查看解答之前先自己嘗試這些練習。 把程式碼打出來而不是複製貼上 — 肌肉記憶有幫助。

練習 1：年齡組分類器

寫一個程式，接受一個 age 變數並印出 該人屬於哪個年齡組：

  • 0–12：「Child」
  • 13–17：「Teenager」
  • 18–64：「Adult」
  • 65+：「Senior」
顯示解答 
age = 30

if age < 0:
    print("Invalid age")
elif age <= 12:
    print("Child")
elif age <= 17:
    print("Teenager")
elif age <= 64:
    print("Adult")
else:
    print("Senior")

# 輸出：Adult

練習 2：溫度建議

給定攝氏溫度 temp，印出建議：

  • 低於 0：「Freezing! Stay inside.」
  • 0–15：「Cold. Wear a jacket.」
  • 16–25：「Nice weather. Enjoy!」
  • 26–35：「Warm. Stay hydrated.」
  • 高於 35：「Dangerously hot!」
顯示解答 
temp = 22

if temp < 0:
    print("Freezing! Stay inside.")
elif temp <= 15:
    print("Cold. Wear a jacket.")
elif temp <= 25:
    print("Nice weather. Enjoy!")
elif temp <= 35:
    print("Warm. Stay hydrated.")
else:
    print("Dangerously hot!")

# 輸出：Nice weather. Enjoy!

練習 3：密碼強度檢查器

給定一個 password 字串，檢查其強度：

  • 少於 6 個字元：「Weak」
  • 6–11 個字元：「Medium」
  • 12 個或更多字元：「Strong」
  • 加分：如果密碼包含至少一個數字，在輸出後加上「 (has number)」。
顯示解答 
password = "hello2world"

length = len(password)
has_digit = any(char.isdigit() for char in password)

if length < 6:
    strength = "Weak"
elif length < 12:
    strength = "Medium"
else:
    strength = "Strong"

if has_digit:
    strength += " (has number)"

print(f"Password strength: {strength}")
# 輸出：Password strength: Medium (has number)

練習 4：三元練習

使用三元表達式將以下程式碼改寫為單行： 

x = 42
if x % 2 == 0:
    parity = "even"
else:
    parity = "odd"
顯示解答 
x = 42
parity = "even" if x % 2 == 0 else "odd"
print(parity)  # 輸出：even

總結

  • if 檢查條件；縮排區塊僅在條件為 True 時執行。
  • elif（不是 elseif！）提供額外的分支。
  • else 是兜底選項，在沒有條件匹配時執行。
  • 每個條件行以冒號結尾。
  • 縮排定義區塊 — 沒有 end 關鍵字。
  • 使用 and、or 和 not 來組合條件。
  • 三元語法：value = a if condition else b。
  • Python 的值具有固有的真值性 — 空集合和零是假值。
  • 使用 == 進行比較，不是 =。始終以冒號結尾。一致地縮排。
