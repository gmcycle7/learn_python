函式與 Lambda

將程式碼組織成可重複使用的區塊。學習如何定義函式、 傳遞參數、回傳值，以及使用匿名 lambda 運算式。

MATLAB 使用者注意：在 MATLAB 中，函式通常 存放在各自的 .m 檔案中（或自 R2016b 起可放在腳本末尾）。 Python 更加靈活 — 你可以在腳本的任何位置定義函式， 而且可以在單一檔案中放置任意數量的函式。MATLAB 使用 function 關鍵字；Python 使用 def。Python 中的回傳值是顯式的 （你需要寫 return），而 MATLAB 則是將值指定給函式簽名中 宣告的輸出變數。

1. 定義函式

函式是一個具名的程式碼區塊，用來執行 特定的任務。你只需定義一次，然後可以根據需要 呼叫它任意多次。這避免了重複相同的程式碼，並使 你的程式更容易閱讀、測試和維護。

語法

def function_name(parameters):
    """選擇性的文件字串。"""
    # 函式主體 — 縮排的程式碼
    return value  # 選擇性
  • def — 每個函式定義開頭的關鍵字。
  • function_name — 使用 snake_case（小寫加底線）。描述性的名稱最佳：calculate_area，而非 ca。
  • 括號 — 用來放置參數（輸入）。如果沒有輸入則使用空的 ()。
  • 冒號 — 在 def 行的結尾是必要的。
  • 縮排主體 — 函式內部的所有內容都必須縮排（通常是 4 個空格）。

範例 1 — 簡單的函式

# 定義函式
def greet():
    print("Hello, welcome to Python!")

# 呼叫它
greet()          # Hello, welcome to Python!
greet()          # Hello, welcome to Python!

範例 2 — 帶參數的函式

def greet_user(name):
    print(f"Hello, {name}! Welcome to Python.")

greet_user("Alice")   # Hello, Alice! Welcome to Python.
greet_user("Bob")     # Hello, Bob! Welcome to Python.

範例 3 — 執行計算的函式

def circle_area(radius):
    pi = 3.14159
    area = pi * radius ** 2
    return area

result = circle_area(5)
print(result)   # 78.53975
MATLAB 對照： 
% MATLAB — 函式在獨立的 circle_area.m 檔案中
function area = circle_area(radius)
    area = pi * radius^2;
end

在 MATLAB 中，你需要將此函式儲存在名為 circle_area.m 的檔案中。在 Python 中，函式可以直接 放在你的腳本中與其他程式碼一起。不需要特殊的檔案。

常見錯誤：忘記在 def 行結尾加冒號， 或忘記縮排函式主體。兩者都會導致 SyntaxError。 
# 錯誤 — 缺少冒號
def greet()
    print("Hi")    # SyntaxError!

# 錯誤 — 沒有縮排
def greet():
print("Hi")        # IndentationError!

2. 參數與引數

參數（parameter）和引數（argument） 這兩個術語經常互換使用，但有一個微妙的差別：

  • 參數 — 函式定義中的變數名稱（佔位符）。
  • 引數 — 呼叫函式時傳入的實際值。
def add(a, b):      # a 和 b 是參數
    return a + b

result = add(3, 5)  # 3 和 5 是引數

位置引數

預設情況下，引數是按位置（從左到右） 與參數配對的： 

def describe_pet(animal, name):
    print(f"I have a {animal} named {name}.")

describe_pet("cat", "Whiskers")   # I have a cat named Whiskers.
describe_pet("Whiskers", "cat")   # I have a Whiskers named cat.  （錯誤！）
注意：使用位置引數時順序很重要！ 交換它們會得到邏輯上錯誤（但語法上有效）的結果。

關鍵字引數

你可以用名稱指定引數，這樣順序就不重要了： 

describe_pet(name="Whiskers", animal="cat")
# I have a cat named Whiskers.  （無論順序如何都正確）

預設值

你可以為參數設定預設值。如果呼叫者未提供 該引數，則使用預設值： 

def greet(name, greeting="Hello"):
    print(f"{greeting}, {name}!")

greet("Alice")               # Hello, Alice!
greet("Alice", "Good morning")  # Good morning, Alice!
規則：有預設值的參數必須放在 沒有預設值的參數之後。 
# 錯誤 — 預設值在非預設值之前
def greet(greeting="Hello", name):   # SyntaxError!
    print(f"{greeting}, {name}!")

範例 — 結合位置、關鍵字和預設值

def create_profile(name, age, city="Unknown", hobby="coding"):
    print(f"{name}, age {age}, from {city}, enjoys {hobby}.")

create_profile("Alice", 30)
# Alice, age 30, from Unknown, enjoys coding.

create_profile("Bob", 25, city="London")
# Bob, age 25, from London, enjoys coding.

create_profile("Carol", 28, hobby="painting", city="Tokyo")
# Carol, age 28, from Tokyo, enjoys painting.

3. 回傳值

return 陳述式將結果送回給呼叫者。 如果沒有 return，函式會回傳 None。

回傳單一值

def square(n):
    return n ** 2

result = square(7)
print(result)   # 49

回傳多個值（元組解包）

Python 可以回傳用逗號分隔的多個值。底層會回傳一個 元組，你可以將其解包到不同的變數中： 

def min_max(numbers):
    return min(numbers), max(numbers)

low, high = min_max([4, 1, 8, 3])
print(low)    # 1
print(high)   # 8
MATLAB 對照：MATLAB 使用方括號語法來取得 多個輸出：[low, high] = minmax(data)。Python 使用元組解包，感覺非常相似。

回傳 None

如果函式沒有 return 陳述式（或使用了 不帶值的 return），它會回傳 None： 

def say_hello(name):
    print(f"Hello, {name}!")
    # 沒有 return 陳述式

result = say_hello("Alice")   # Hello, Alice!
print(result)                  # None
常見錯誤：混淆 print() 和 return。print() 在螢幕上顯示文字，但 不會將值送回給呼叫者。return 會將值送回，但不會顯示任何東西。 
def bad_add(a, b):
    print(a + b)       # 顯示 8，但回傳 None

def good_add(a, b):
    return a + b       # 回傳 8，由呼叫者決定如何處理

x = bad_add(3, 5)     # 印出 8，但 x 是 None！
y = good_add(3, 5)    # y 是 8

提前回傳

你可以使用 return 來提前退出函式： 

def divide(a, b):
    if b == 0:
        return "Error: division by zero"
    return a / b

print(divide(10, 3))   # 3.3333333333333335
print(divide(10, 0))   # Error: division by zero

4. 作用域 — 區域 vs. 全域

作用域決定了變數可以被看到和使用的範圍。 在 Python 中，每個函式都會建立自己的區域作用域。

區域變數

在函式內部建立的變數僅存在於該函式內部： 

def my_function():
    secret = 42        # 區域變數
    print(secret)

my_function()           # 42
print(secret)           # NameError: name 'secret' is not defined

全域變數

在任何函式外部建立的變數是全域的 — 它們可以在函式內部被讀取，但如果沒有 global 關鍵字就無法修改： 

greeting = "Hello"     # 全域

def say_hi():
    print(greeting)    # 讀取全域變數 — 沒問題

say_hi()               # Hello
counter = 0

def increment():
    counter = counter + 1   # 錯誤！Python 認為 counter 是區域變數
                            # 因為你正在對它進行賦值

increment()   # UnboundLocalError

global 關鍵字（謹慎使用！）

counter = 0

def increment():
    global counter         # 告訴 Python：使用全域變數
    counter = counter + 1

increment()
increment()
print(counter)   # 2
最佳實踐：在大多數情況下避免使用 global。 它會使程式碼更難理解、測試和除錯，因為任何函式都可能 暗中改變一個變數。相反，將值作為引數傳遞並回傳結果： 
# 更好的方法 — 不需要 global 關鍵字
def increment(counter):
    return counter + 1

count = 0
count = increment(count)   # 1
count = increment(count)   # 2

LEGB 規則（簡化版）

當 Python 遇到一個變數名稱時，會按以下順序搜尋：

  1. Local（區域） — 在目前的函式內部。
  2. Enclosing（外圍） — 在任何外圍（外層）函式內部（用於巢狀函式）。
  3. Global（全域） — 在模組（檔案）層級。
  4. Built-in（內建） — Python 的內建名稱（print、len 等）。
x = "global"

def outer():
    x = "enclosing"

    def inner():
        x = "local"
        print(x)       # local

    inner()
    print(x)           # enclosing

outer()
print(x)               # global
初學者提示：你主要需要記住兩件事： （1）函式內部的變數是區域的，（2）如果 Python 無法在區域 找到變數，它會向外層搜尋。完整的 LEGB 規則在你開始 撰寫巢狀函式時會變得重要。

5. 文件字串

文件字串（docstring）是出現在函式第一個陳述式位置的 字串常量。它記錄了函式的功能。 使用三重引號（"""..."""）來撰寫文件字串： 

def celsius_to_fahrenheit(celsius):
    """將溫度從攝氏轉換為華氏。

    Args:
        celsius: 攝氏溫度。

    Returns:
        華氏溫度。
    """
    return celsius * 9 / 5 + 32

為什麼要寫文件字串？

  • 你（或其他人）可以在數個月後閱讀文件字串來了解函式的功能。
  • 內建的 help() 函式會自動顯示文件字串：
help(celsius_to_fahrenheit)

# 輸出：
# Help on function celsius_to_fahrenheit in module __main__:
#
# celsius_to_fahrenheit(celsius)
#     將溫度從攝氏轉換為華氏。
#     ...

直接存取文件字串

print(celsius_to_fahrenheit.__doc__)
# 將溫度從攝氏轉換為華氏。
# ...
MATLAB 對照：在 MATLAB 中，你在函式檔案頂部 以註解的方式撰寫說明文字。Python 使用專門的 文件字串（一個真正的字串，而非註解），工具可以自動解析它。 概念是相同的：為你的函式撰寫文件！

範例 — 良好的文件字串習慣

def calculate_bmi(weight_kg, height_m):
    """計算身體質量指數（BMI）。

    Args:
        weight_kg: 體重（公斤）。
        height_m: 身高（公尺）。

    Returns:
        BMI 值（浮點數）。

    Example:
        >>> calculate_bmi(70, 1.75)
        22.857142857142858
    """
    return weight_kg / height_m ** 2

6. Lambda 函式

lambda 是一個小型的匿名函式（沒有名稱的函式）。 它使用 lambda 關鍵字在單行中定義： 

# 語法：lambda 參數: 運算式

double = lambda x: x * 2
print(double(5))    # 10

add = lambda a, b: a + b
print(add(3, 7))    # 10

Lambda 與 def 的比較

# 一般函式
def square(x):
    return x ** 2

# Lambda 等價寫法
square = lambda x: x ** 2

# 兩者的用法相同
print(square(4))   # 16

何時使用 Lambda

Lambda 最適合作為短小的、一次性的函式傳遞給 其他函式，如 sorted()、map() 或 filter()： 

# 按第二個元素排序元組串列
pairs = [(1, "b"), (3, "a"), (2, "c")]
sorted_pairs = sorted(pairs, key=lambda x: x[1])
print(sorted_pairs)   # [(3, 'a'), (1, 'b'), (2, 'c')]
# 使用 map() 將每個數字加倍
numbers = [1, 2, 3, 4, 5]
doubled = list(map(lambda x: x * 2, numbers))
print(doubled)   # [2, 4, 6, 8, 10]
# 使用 filter() 只保留偶數
evens = list(filter(lambda x: x % 2 == 0, numbers))
print(evens)   # [2, 4]

何時不該使用 Lambda

避免在以下情況使用 lambda：
  • 邏輯複雜或超過一個運算式 — 改用一般的 def 函式。
  • 你需要多次重複使用該函式 — 用 def 給它一個正式的名稱。
  • 你需要文件字串 — lambda 無法擁有文件字串。
  • 可讀性受損 — 如果你需要瞇著眼睛才能理解它，就用 def。
# 不好 — 對 lambda 來說太複雜了
process = lambda x: x.strip().lower().replace(" ", "_") if isinstance(x, str) else str(x)

# 更好 — 使用 def
def process(x):
    """清理一個值以作為鍵值使用。"""
    if isinstance(x, str):
        return x.strip().lower().replace(" ", "_")
    return str(x)

7. 函式作為一等物件

在 Python 中，函式是一等物件。這代表 你可以：

  • 將函式指定給變數。
  • 將函式作為引數傳遞給另一個函式。
  • 從另一個函式回傳一個函式。

將函式指定給變數

def shout(text):
    return text.upper() + "!"

yell = shout              # yell 現在參照相同的函式
print(yell("hello"))      # HELLO!

將函式作為引數傳遞

def apply_twice(func, value):
    """對一個值套用函式兩次。"""
    return func(func(value))

def add_three(x):
    return x + 3

result = apply_twice(add_three, 10)
print(result)   # 16  (10 + 3 = 13，然後 13 + 3 = 16)

實用範例 — 自訂排序

students = [
    {"name": "Alice", "grade": 88},
    {"name": "Bob", "grade": 95},
    {"name": "Carol", "grade": 72},
]

def get_grade(student):
    return student["grade"]

# 將函式作為排序的 key 傳入
ranked = sorted(students, key=get_grade, reverse=True)
for s in ranked:
    print(f"{s['name']}: {s['grade']}")
# Bob: 95
# Alice: 88
# Carol: 72
重要觀念：當你將函式作為引數傳遞時， 寫函式名稱時不要加括號： key=get_grade，而非 key=get_grade()。 加括號會立即呼叫該函式，這不是你想要的。

8. *args 與 **kwargs

有時你想要一個接受任意數量引數的函式。 Python 為此提供了兩種特殊語法。

*args — 可變位置引數

*args 參數會將額外的位置引數收集到一個 元組中： 

def add_all(*args):
    """將任意數量的值加在一起。"""
    print(f"Received: {args}")     # args 是一個元組
    return sum(args)

print(add_all(1, 2))           # Received: (1, 2) → 3
print(add_all(1, 2, 3, 4, 5)) # Received: (1, 2, 3, 4, 5) → 15

範例 — 靈活的問候

def greet_all(*names):
    for name in names:
        print(f"Hello, {name}!")

greet_all("Alice", "Bob", "Carol")
# Hello, Alice!
# Hello, Bob!
# Hello, Carol!

**kwargs — 可變關鍵字引數

**kwargs 參數會將額外的關鍵字引數收集到一個 字典中： 

def build_profile(name, **kwargs):
    """建立一個使用者資料字典。"""
    profile = {"name": name}
    profile.update(kwargs)
    return profile

user = build_profile("Alice", age=30, city="London", job="engineer")
print(user)
# {'name': 'Alice', 'age': 30, 'city': 'London', 'job': 'engineer'}

結合 *args 與 **kwargs

def super_function(*args, **kwargs):
    print(f"Positional: {args}")
    print(f"Keyword:    {kwargs}")

super_function(1, 2, 3, name="Alice", age=30)
# Positional: (1, 2, 3)
# Keyword:    {'name': 'Alice', 'age': 30}
命名慣例：args 和 kwargs 只是慣例而已。你可以使用 *values 和 **options — 重要的是 * 和 ** 前綴。然而，強烈建議堅持使用 args 和 kwargs，因為每個 Python 開發者都能立即辨認它們。

參數順序規則

當組合所有參數類型時，它們必須按以下順序出現： 

def example(pos1, pos2, default="x", *args, **kwargs):
    pass

# 1. 一般位置參數
# 2. 具有預設值的參數
# 3. *args
# 4. **kwargs
MATLAB 對照：MATLAB 使用 varargin 處理可變長度的輸入引數，並使用 nargin 檢查 傳入了多少引數。Python 的 *args 和 **kwargs 更加優雅和明確 — 你可以 直接迭代它們，而不需要檢查引數數量。

練習題

練習你所學到的內容。在查看答案之前，先自己嘗試每個練習。

練習 1 — 矩形面積

撰寫一個函式 rectangle_area(width, height)， 回傳矩形的面積。用幾個不同的值來測試它。

顯示解答 
def rectangle_area(width, height):
    """回傳矩形的面積。"""
    return width * height

print(rectangle_area(5, 3))    # 15
print(rectangle_area(10, 2))   # 20
print(rectangle_area(7.5, 4))  # 30.0
練習 2 — 溫度轉換器

撰寫一個函式 f_to_c(fahrenheit)，使用公式 C = (F - 32) * 5/9 將華氏轉換為攝氏。 回傳四捨五入到小數點後 2 位的結果。

顯示解答 
def f_to_c(fahrenheit):
    """將華氏轉換為攝氏。"""
    celsius = (fahrenheit - 32) * 5 / 9
    return round(celsius, 2)

print(f_to_c(32))    # 0.0
print(f_to_c(212))   # 100.0
print(f_to_c(98.6))  # 37.0
練習 3 — 預設問候語

撰寫一個函式 greet(name, greeting="Hello")， 印出 "{greeting}, {name}!"。分別在有和沒有 greeting 引數的情況下呼叫它。

顯示解答 
def greet(name, greeting="Hello"):
    """印出個人化的問候語。"""
    print(f"{greeting}, {name}!")

greet("Alice")                # Hello, Alice!
greet("Bob", "Good morning")  # Good morning, Bob!
練習 4 — 回傳多個值

撰寫一個函式 stats(numbers)，接收一個數字串列 並回傳三個值：最小值、最大值和平均值。將它們解包到不同的變數中。

顯示解答 
def stats(numbers):
    """回傳串列的最小值、最大值和平均值。"""
    minimum = min(numbers)
    maximum = max(numbers)
    average = sum(numbers) / len(numbers)
    return minimum, maximum, average

low, high, avg = stats([4, 8, 15, 16, 23, 42])
print(f"Min: {low}, Max: {high}, Avg: {avg:.2f}")
# Min: 4, Max: 42, Avg: 18.00
練習 5 — Lambda 排序

給定串列 words = ["banana", "pie", "Washington", "a"]， 使用 sorted() 搭配 lambda 按單字的 長度排序（最短的在前面）。

顯示解答 
words = ["banana", "pie", "Washington", "a"]
by_length = sorted(words, key=lambda w: len(w))
print(by_length)   # ['a', 'pie', 'banana', 'Washington']
練習 6 — *args 求和

撰寫一個函式 multiply_all(*args)，將所有引數 相乘並回傳結果。如果沒有給引數，回傳 1。

顯示解答 
def multiply_all(*args):
    """將所有引數相乘。"""
    result = 1
    for num in args:
        result *= num
    return result

print(multiply_all(2, 3, 4))    # 24
print(multiply_all(5, 10))      # 50
print(multiply_all())            # 1
練習 7 — **kwargs 個人資料建立器

撰寫一個函式 make_profile(name, **kwargs)， 建立並回傳一個字典。字典應該始終有一個 "name" 鍵， 加上任何傳入的額外關鍵字引數。 用 age、city 和 hobby 來測試它。

顯示解答 
def make_profile(name, **kwargs):
    """從關鍵字引數建立個人資料字典。"""
    profile = {"name": name}
    for key, value in kwargs.items():
        profile[key] = value
    return profile

p = make_profile("Alice", age=30, city="London", hobby="chess")
print(p)
# {'name': 'Alice', 'age': 30, 'city': 'London', 'hobby': 'chess'}
練習 8 — 文件字串練習

撰寫一個函式 is_even(n)，如果 n 是偶數則回傳 True，否則回傳 False。 加上包含 Args、Returns 和 Example 區段的正式文件字串。 然後呼叫 help(is_even) 來驗證文件字串是否正確顯示。

顯示解答 
def is_even(n):
    """檢查一個數字是否為偶數。

    Args:
        n: 要檢查的整數。

    Returns:
        如果 n 是偶數回傳 True，否則回傳 False。

    Example:
        >>> is_even(4)
        True
        >>> is_even(7)
        False
    """
    return n % 2 == 0

print(is_even(4))    # True
print(is_even(7))    # False
help(is_even)
練習 9 — 傳遞函式

撰寫一個函式 apply_to_list(func, lst)， 將給定的函式套用到串列中的每個元素並回傳新的串列。 傳入一個將數字乘以三倍的函式來測試它。

顯示解答 
def apply_to_list(func, lst):
    """對每個元素套用 func 並回傳新串列。"""
    return [func(item) for item in lst]

def triple(x):
    return x * 3

numbers = [1, 2, 3, 4, 5]
result = apply_to_list(triple, numbers)
print(result)   # [3, 6, 9, 12, 15]

# 也可以搭配 lambda 使用：
result2 = apply_to_list(lambda x: x ** 2, numbers)
print(result2)  # [1, 4, 9, 16, 25]
練習 10 — 作用域挑戰

不執行程式碼，預測以下程式會印出什麼。然後執行它來 驗證你的答案： 

x = 10

def outer():
    x = 20
    def inner():
        x = 30
        print("inner:", x)
    inner()
    print("outer:", x)

outer()
print("global:", x)
顯示解答 
# 輸出：
# inner: 30
# outer: 20
# global: 10

# 每個函式都有自己的區域 x。
# 全域的 x 從未被修改。

總結

概念 語法 / 重點
定義函式 def name(params): body
位置引數 按順序配對：func(a, b)
關鍵字引數 按名稱配對：func(name="Alice")
預設值 def f(x, y=10):
回傳值 return value（或多個：return a, b）
作用域 LEGB：區域 > 外圍 > 全域 > 內建
文件字串 函式主體第一行的三重引號字串
Lambda lambda x: expression — 用於短小的一次性函式
一等函式 將函式作為引數傳遞、指定給變數
*args 將額外的位置引數收集到元組中
**kwargs 將額外的關鍵字引數收集到字典中
為什麼這很重要：函式是組織良好的程式碼的基本構建模組。 一旦你能自在地定義和呼叫函式，你就能將任何複雜的問題分解成 小型、可管理、可測試的部分。你撰寫的每一個正式的 Python 程式 都會建立在函式之上。
