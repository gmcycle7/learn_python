Python 術語表

每位初學者都應該知道的 Python 基本術語 A-Z 參考。

ABCDEFGHIJKLMNOPRSTVWZ
A
引數（Argument）
呼叫函式時傳入的值。在 print("hello") 中，"hello" 就是引數。
賦值（Assignment）
使用 = 運算子將值儲存到變數中。例如：x = 10 將整數 10 賦值給變數 x。
assert
一種除錯語句，測試條件是否成立，若為假則觸發 AssertionError。例如：assert len(my_list) > 0 確保串列不為空。
屬性（Attribute）
與物件關聯的值或函式，透過點號語法存取。在 my_list.append(5) 中，append 是串列物件的屬性（具體來說是方法）。
B
布林值（Boolean）
只有兩個可能值的資料型別：True 或 False。用於條件和比較，例如 is_active = True。
break
立即退出最內層迴圈的語句。例如：for n in range(100): if n == 5: break 當 n 到達 5 時停止迴圈。
內建（Built-in）
在 Python 中不需要匯入即可使用的函式或型別。例如 print()、len()、int() 和 list()。
位元組（Byte）
資料的基本單位。Python 有 bytes 型別用於位元組序列，常用於讀取二進位檔案或處理網路資料。使用 b"hello" 建立。
C
類別（Class）
建立物件的藍圖。使用 class 關鍵字定義。例如：class Dog: pass 定義了一個名為 Dog 的簡單類別。
註解（Comment）
程式碼中 Python 會忽略的文字，用於解釋程式碼的功能。單行註解以 # 開頭。例如：# 這是一個註解。
推導式（Comprehension）
建立串列、集合或字典的簡潔方式。例如：[x * 2 for x in range(5)] 產生 [0, 2, 4, 6, 8]。
串接（Concatenation）
將兩個序列連接在一起。字串："Hello" + " World" 得到 "Hello World"。串列：[1, 2] + [3, 4] 得到 [1, 2, 3, 4]。
條件式（Conditional）
僅在條件為真時才執行程式碼的語句。Python 使用 if、elif 和 else 進行條件邏輯判斷。
建構子（Constructor）
特殊方法 __init__，在從類別建立新物件時自動執行。例如：def __init__(self, name): self.name = name。
continue
跳過當前迴圈迭代的剩餘部分並進入下一次迭代的語句。例如：for n in range(5): if n == 2: continue 在 n 為 2 時跳過該次迭代。
CSV
逗號分隔值，一種用於表格資料的純文字檔案格式。Python 的內建 csv 模組可以透過 import csv 輕鬆讀寫 CSV 檔案。
D
資料型別（Data type）
值的類別，決定了可以對其執行哪些操作。常見型別包括 int、float、str、list、dict 和 bool。
裝飾器（Decorator）
修改另一個函式行為的函式。在函式定義上方使用 @ 符號套用。例如：在方法前加上 @staticmethod 使其成為靜態方法。
def
用於定義函式的關鍵字。例如：def greet(name): return f"Hello, {name}"。
預設參數（Default parameter）
函式參數中預先指定的值，當未提供引數時使用。例如：def greet(name="World") 在未給名稱時使用 "World"。
字典（Dictionary）
儲存鍵值對的內建資料結構。使用大括號建立：{"name": "Alice", "age": 30}。透過鍵存取值：d["name"]。
文件字串（Docstring）
放在函式、類別或模組第一行的字串，用於文件記錄。使用三引號撰寫："""此函式執行 X。"""。可透過 help() 存取。
動態型別（Dynamic typing）
Python 的特性，變數沒有固定型別。你可以寫 x = 5 然後 x = "hello"，Python 不會抱怨——型別在執行時改變。
E
元素（Element）
集合（如串列、元組或集合）中的單個項目。在 [10, 20, 30] 中，20 是索引 1 的元素。
elif
「else if」的縮寫。在 if 區塊後用於檢查另一個條件。例如：elif x > 0:。
else
當前面的 if（和所有 elif）條件都為假時執行的子句。例如：else: print("No match")。
封裝（Encapsulation）
物件導向原則，將資料和方法綁定在類別內部，並限制對某些內部元素的直接存取。在 Python 中，前導底線（例如 _private）表示屬性是內部的。
enumerate()
為可迭代物件添加索引計數器的內建函式。例如：for i, val in enumerate(["a", "b"]) 產生 (0, "a") 和 (1, "b") 對。
例外（Exception）
程式執行期間發生的錯誤。常見例外包括 ValueError、TypeError 和 IndexError。使用 try/except 處理。
運算式（Expression）
計算後會產生值的程式碼片段。例如：3 + 4、len("hi") 和 x > 5 都是運算式。
F
f-字串（f-string）
格式化字串字面值（Python 3.6+），可在大括號內嵌入運算式。例如：f"Hello, {name}!" 將 name 的值插入字串。
False
兩個布林值之一。在數值上下文中等同於 0。0、""、[] 和 None 等值都是「假值」。
filter()
回傳可迭代物件中函式回傳 True 的元素的內建函式。例如：list(filter(lambda x: x > 0, [-1, 2, 3])) 得到 [2, 3]。
finally
try/except 語句中無論是否發生例外都會執行的區塊。常用於清理工作，如關閉檔案。
浮點數（Float）
帶小數點的數字。例如：3.14。可用 float("3.14") 或直接寫小數字面值建立。
for 迴圈
遍歷序列中每個項目的迴圈。例如：for fruit in ["apple", "banana"]: print(fruit)。
函式（Function）
使用 def 定義的可重複使用的程式碼區塊，執行特定任務。透過名稱和括號呼叫：greet("Alice")。
G
產生器（Generator）
使用 yield 而非 return 來延遲產生值序列的函式（一次一個）。例如：def count(): yield 1; yield 2。
global
允許函式修改模組層級變數的關鍵字。例如：函式內的 global counter 允許修改外部的 counter 變數。
全域變數（Global variable）
在腳本或模組頂層定義的變數，可在該檔案的任何地方存取。請謹慎使用——優先將值作為函式引數傳遞。
H
雜湊（Hash）
從物件計算出的固定大小數值，用於字典和集合中的快速查找。只有不可變型別（如 str、int 和 tuple）是可雜湊的。
help()
顯示任何物件文件的內建函式。在直譯器中輸入 help(print) 即可查看 print() 的用法。
I
IDE
整合開發環境——用於撰寫程式碼的軟體應用程式，具有語法高亮、除錯和自動完成等功能。常用的 Python IDE 包括 VS Code 和 PyCharm。
if
開始條件語句的關鍵字。縮排區塊僅在條件為真時執行。例如：if x > 0: print("positive")。
不可變（Immutable）
建立後值無法更改的物件。字串、整數、浮點數和元組在 Python 中是不可變的。例如：你無法執行 "hello"[0] = "H"。
匯入（Import）
import 語句載入模組，使你可以使用其函式和類別。例如：import math 然後 math.sqrt(16)。
索引（Index）
元素在序列中的位置，從 0 開始。在 ["a", "b", "c"] 中，"b" 的索引是 1。負索引從末尾計算：-1 是最後一個項目。
繼承（Inheritance）
物件導向機制，一個類別（子類別）繼承另一個類別（父類別）的屬性和方法。例如：class Cat(Animal): 表示 Cat 繼承自 Animal。
input()
從使用者讀取一行文字的內建函式。例如：name = input("Enter your name: ")。總是回傳字串。
實例（Instance）
從類別建立的特定物件。如果 Dog 是一個類別，那麼 my_dog = Dog("Rex") 建立了 Dog 的一個實例。
整數（Integer）
沒有小數點的整數，如 42 或 -7。Python 整數具有無限精度。
直譯器（Interpreter）
逐行讀取和執行 Python 程式碼的程式。你可以在終端機輸入 python 以互動方式執行，或用它執行腳本如 python script.py。
可迭代物件（Iterable）
任何可以用 for 迴圈遍歷的物件。串列、元組、字串、字典、集合和產生器都是可迭代物件。
迭代器（Iterator）
當你對其呼叫 next() 時產生序列中下一個值的物件。使用 iter() 從可迭代物件建立。例如：it = iter([1, 2]); next(it) 得到 1。
J
JSON
JavaScript 物件標記法——一種輕量級的文字格式，用於儲存和交換資料。Python 的 json 模組可以處理：json.loads('{"key": "value"}') 將 JSON 解析為字典。
join()
將字串串列合併為一個字串的字串方法，使用該字串作為分隔符。例如：", ".join(["a", "b", "c"]) 得到 "a, b, c"。
K
鍵（Key）
用於存取字典中值的識別符。在 {"name": "Alice"} 中，"name" 是鍵。鍵必須是不可變的（字串、數字、元組）。
關鍵字（Keyword）
Python 中具有特殊含義且不能用作變數名稱的保留字。例如：if、for、def、class、return、import。
關鍵字引數（Keyword argument）
透過名稱傳遞給函式的引數。例如：print("hi", end="")——這裡 end="" 是關鍵字引數。
L
Lambda
使用 lambda 關鍵字在單一運算式中定義的匿名（無名）函式。例如：square = lambda x: x ** 2 然後 square(4) 得到 16。
len()
回傳物件中項目數量的內建函式。例如：len([1, 2, 3]) 得到 3；len("hello") 得到 5。
函式庫（Library）
提供預先撰寫功能的模組和套件集合。例如：Python 標準函式庫、NumPy、Pandas 和 Requests。
串列（List）
使用方括號建立的有序、可變集合。例如：fruits = ["apple", "banana", "cherry"]。透過索引存取項目：fruits[0]。
串列推導式（List comprehension）
建立串列的精簡語法。例如：[x ** 2 for x in range(5)] 得到 [0, 1, 4, 9, 16]。可以包含條件：[x for x in range(10) if x % 2 == 0]。
區域變數（Local variable）
在函式內部定義的變數，僅在該函式內可存取。它在函式執行時建立，函式結束時銷毀。
迴圈（Loop）
重複執行程式碼區塊的控制結構。Python 有兩種：for 迴圈（遍歷序列）和 while 迴圈（在條件為真時重複）。
M
map()
將函式套用到可迭代物件中每個項目的內建函式。例如：list(map(str, [1, 2, 3])) 得到 ["1", "2", "3"]。
方法（Method）
屬於物件的函式，使用點號語法呼叫。例如："hello".upper() 呼叫字串 "hello" 的 upper 方法，回傳 "HELLO"。
模組（Module）
包含函式、類別和變數的單一 Python 檔案（.py），可以被匯入使用。例如：import math 匯入 math 模組。
可變（Mutable）
建立後值可以更改的物件。串列、字典和集合是可變的。例如：my_list[0] = 99 原地更改第一個元素。
N
None
Python 的空值，表示沒有值。未明確回傳的函式會回傳 None。使用 if x is None: 進行檢查。
NumPy
流行的第三方數值計算函式庫，具有強大的陣列操作功能。使用 pip install numpy 安裝。例如：import numpy as np; np.array([1, 2, 3])。
O
物件（Object）
Python 中的一切都是物件——一份具有型別、值和相關方法的資料。整數、字串、串列、函式和類別都是物件。
open()
開啟檔案並回傳檔案物件的內建函式。例如：with open("data.txt", "r") as f: content = f.read()。
運算子（Operator）
對值執行操作的符號。算術：+、-、*、/。比較：==、!=、<、>。邏輯：and、or、not。
P
套件（Package）
以 __init__.py 檔案組織的 Python 模組目錄。套件讓你組織大型專案。例如：import mypackage.mymodule。
Pandas
強大的第三方資料分析和處理函式庫，建立在 NumPy 之上。使用 pip install pandas 安裝。其核心資料結構是 DataFrame。
參數（Parameter）
函式定義中列出的變數，在函式被呼叫時接收值。在 def greet(name): 中，name 是參數。與引數（傳遞的實際值）相對。
pass
不做任何事的語句。當語法上需要程式碼區塊但你還沒有要寫的內容時用作佔位符。例如：def todo(): pass。
PEP 8
Python 官方風格指南。建議使用 4 個空格縮排、變數和函式使用 snake_case，類別名稱使用 PascalCase 等慣例。
pip
Python 的套件安裝工具。用於從 PyPI 安裝第三方函式庫。例如：pip install requests 安裝 Requests 函式庫。
print()
將輸出顯示到主控台的內建函式。例如：print("Hello, World!")。接受多個引數：print("a", "b", sep="-") 輸出 a-b。
屬性裝飾器（Property）
使用 @property 裝飾器在類別屬性上定義 getter 和 setter 方法的方式。讓你像存取簡單屬性一樣存取方法。
Python
由 Guido van Rossum 創建的高階、直譯式、通用程式語言。以可讀的語法、龐大的標準函式庫和廣泛的社群支援著稱。
R
range()
產生數字序列的內建函式。例如：range(5) 產生 0, 1, 2, 3, 4。常用於 for 迴圈：for i in range(3):。
遞迴（Recursion）
函式透過將問題拆分為更小的子問題來呼叫自身。例如：def factorial(n): return 1 if n == 0 else n * factorial(n - 1)。
REPL
讀取-求值-輸出迴圈——互動式 Python 殼層，你輸入運算式後立即看到結果。在終端機執行 python 即可啟動。
return
退出函式並將值傳回給呼叫者的語句。例如：def add(a, b): return a + b。沒有 return 的函式隱式回傳 None。
S
作用域（Scope）
變數可存取的程式碼區域。Python 使用 LEGB 規則：區域（Local）、外層（Enclosing）、全域（Global）、內建（Built-in）。在函式內定義的變數具有區域作用域。
self
類別中實例方法第一個參數的慣用名稱。它指向目前的物件。例如：def greet(self): print(self.name)。
集合（Set）
不重複元素的無序集合，使用大括號或 set() 建立。例如：{1, 2, 3}。適合去除重複和集合操作如聯集和交集。
切片（Slice）
使用 [start:stop:step] 語法從序列中擷取部分的方式。例如："Python"[0:3] 得到 "Pyt"；[1, 2, 3, 4][::2] 得到 [1, 3]。
sorted()
從任何可迭代物件回傳新排序串列的內建函式。例如：sorted([3, 1, 2]) 得到 [1, 2, 3]。使用 reverse=True 做降序排列。
標準函式庫（Standard library）
每個 Python 安裝都附帶的模組集合。涵蓋檔案 I/O、數學、JSON、CSV、正規表達式、網路等——不需要 pip install。
字串（String）
以引號包圍的字元序列。例如："Hello" 或 'Hello'。字串是不可變的。使用三引號表示多行字串："""第1行\n第2行"""。
super()
在子類別中呼叫父類別方法的內建函式。例如：super().__init__(name) 呼叫父類別的建構子。
T
追蹤（Traceback）
例外發生時 Python 顯示的錯誤報告。它顯示導致錯誤的函式呼叫鏈，最近的呼叫在最後。從底部往上讀以找到原因。
True
兩個布林值之一。在數值上下文中等同於 1。非空字串、非零數字和非空集合都是「真值」。
try/except
處理例外的語句。嘗試執行 try 區塊中的程式碼；如果發生錯誤，則執行 except 區塊。例如：try: int("abc") except ValueError: print("Not a number")。
元組（Tuple）
使用括號建立的有序、不可變序列。例如：point = (3, 4)。像是無法修改的串列。透過索引存取項目：point[0]。
型別（Type）
值的類別。使用 type() 檢查：type(42) 回傳 <class 'int'>。常見型別：int、float、str、list、dict、bool。
型別轉換（Type conversion）
將值從一種型別改為另一種。例如：int("5") 將字串 "5" 轉為整數 5。也稱為型別轉型。常見轉換：str()、int()、float()、list()。
V
變數（Variable）
儲存在記憶體中值的命名參考。透過賦值建立：age = 25。變數名稱應具描述性並遵循 snake_case 慣例。
venv
用於建立虛擬環境的內建模組。用法：python -m venv myenv 在 myenv 資料夾中建立隔離環境。
虛擬環境（Virtual environment）
擁有獨立套件集的隔離 Python 安裝，與系統 Python 分開。防止專案間的版本衝突。使用 python -m venv myenv 建立，使用前需啟用。
W
while 迴圈
在條件保持為真時重複執行程式碼區塊的迴圈。例如：while count < 10: count += 1。注意避免無限迴圈。
with 語句
確保資源被正確設定和清理的上下文管理器。最常與檔案搭配使用：with open("file.txt") as f: 在區塊結束時自動關閉檔案。
Z
零基索引（Zero-based indexing）
Python 的慣例，序列的第一個元素在索引 0，第二個在 1，依此類推。例如：在 ["a", "b", "c"] 中，"a" 在索引 0。
zip()
將多個可迭代物件逐元素組合成元組的內建函式。例如：list(zip([1, 2], ["a", "b"])) 得到 [(1, "a"), (2, "b")]。適合平行遍歷兩個串列。
