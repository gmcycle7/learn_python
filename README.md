# Learn Python — Complete Beginner's Guide

A comprehensive, self-contained website for learning Python from scratch. Designed for beginners, with special attention to learners coming from a MATLAB background.

## How to Open the Website

### Option 1: Just open the file (simplest)
Double-click `index.html` in this folder, or open it in any web browser.

### Option 2: Local server (recommended for best experience)
Open a terminal in this folder and run:

```bash
python -m http.server 8000
```

Then open your browser to: **http://localhost:8000**

1. **What is Python?** — Get the big picture
2. **Getting Started** — Install Python and set up your environment
3. **Print, Variables & Numbers** — Write your first code
4. **Strings & Booleans** — Work with text and logic
5. **Lists, Tuples & Sets** — Store collections of data
6. **Dictionaries** — Key-value data storage
7. **Conditionals** — Make decisions in code

12. **Common Patterns** — Practical recipes you'll use daily
13. **Mini Projects** — Apply what you've learned
14. **Practice & Roadmap** — Exercises and a 30-day plan
