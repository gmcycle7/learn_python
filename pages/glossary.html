Python Glossary

An A-Z reference of essential Python terms every beginner should know.

ABCDEFGHIJKLMNOPRSTVWZ
A
Argument
A value passed to a function when calling it. In print("hello"), "hello" is the argument.
Assignment
Storing a value in a variable using the = operator. Example: x = 10 assigns the integer 10 to the variable x.
assert
A debugging statement that tests a condition and raises AssertionError if it is false. Example: assert len(my_list) > 0 ensures the list is not empty.
Attribute
A value or function associated with an object, accessed with dot notation. In my_list.append(5), append is an attribute (specifically a method) of the list object.
B
Boolean
A data type with only two possible values: True or False. Used in conditions and comparisons, e.g. is_active = True.
break
A statement that immediately exits the innermost loop. Example: for n in range(100): if n == 5: break stops the loop when n reaches 5.
Built-in
A function or type that is always available in Python without needing an import. Examples include print(), len(), int(), and list().
Byte
A unit of data. Python has a bytes type for sequences of bytes, often used when reading binary files or working with network data. Created with b"hello".
C
Class
A blueprint for creating objects. Defined with the class keyword. Example: class Dog: pass defines a simple class named Dog.
Comment
Text in your code that Python ignores, used to explain what the code does. Single-line comments start with #. Example: # This is a comment.
Comprehension
A concise way to create lists, sets, or dictionaries. Example: [x * 2 for x in range(5)] produces [0, 2, 4, 6, 8].
Concatenation
Joining two sequences together. For strings: "Hello" + " World" gives "Hello World". For lists: [1, 2] + [3, 4] gives [1, 2, 3, 4].
Conditional
A statement that runs code only when a condition is true. Python uses if, elif, and else for conditional logic.
Constructor
The special method __init__ that runs automatically when a new object is created from a class. Example: def __init__(self, name): self.name = name.
continue
A statement that skips the rest of the current loop iteration and moves to the next one. Example: for n in range(5): if n == 2: continue skips the iteration when n is 2.
CSV
Comma-Separated Values, a plain-text file format for tabular data. Python's built-in csv module lets you read and write CSV files easily with import csv.
D
Data type
The category of a value that determines what operations can be performed on it. Common types include int, float, str, list, dict, and bool.
Decorator
A function that modifies the behavior of another function. Applied with the @ symbol above a function definition. Example: @staticmethod before a method makes it a static method.
def
The keyword used to define a function. Example: def greet(name): return f"Hello, {name}".
Default parameter
A function parameter with a pre-assigned value that is used when no argument is provided. Example: def greet(name="World") uses "World" if no name is given.
Dictionary
A built-in data structure that stores key-value pairs. Created with curly braces: {"name": "Alice", "age": 30}. Access values by key: d["name"].
Docstring
A string placed as the first statement in a function, class, or module to document it. Written with triple quotes: """This function does X.""". Accessible via help().
Dynamic typing
Python's approach where variables do not have fixed types. You can write x = 5 then x = "hello" and Python will not complain -- the type changes at runtime.
E
Element
A single item within a collection such as a list, tuple, or set. In [10, 20, 30], 20 is an element at index 1.
elif
Short for "else if." Used after an if block to check another condition. Example: elif x > 0:.
else
A clause that runs when the preceding if (and all elif) conditions are false. Example: else: print("No match").
Encapsulation
An object-oriented principle of bundling data and methods together inside a class, and restricting direct access to some internals. In Python, a leading underscore (e.g. _private) signals an attribute is internal.
enumerate()
A built-in that adds an index counter to an iterable. Example: for i, val in enumerate(["a", "b"]) gives pairs (0, "a") and (1, "b").
Exception
An error that occurs during program execution. Common exceptions include ValueError, TypeError, and IndexError. Handle them with try/except.
Expression
A piece of code that evaluates to a value. Examples: 3 + 4, len("hi"), and x > 5 are all expressions.
F
f-string
A formatted string literal (Python 3.6+) that lets you embed expressions inside curly braces. Example: f"Hello, {name}!" inserts the value of name into the string.
False
One of the two Boolean values. Equivalent to 0 in numeric contexts. Values like 0, "", [], and None are all "falsy."
filter()
A built-in that returns elements from an iterable for which a function returns True. Example: list(filter(lambda x: x > 0, [-1, 2, 3])) gives [2, 3].
finally
A block in a try/except statement that always executes, whether or not an exception occurred. Often used for cleanup like closing files.
Float
A number with a decimal point. Example: 3.14. Created with float("3.14") or by writing a decimal literal directly.
for loop
A loop that iterates over each item in a sequence. Example: for fruit in ["apple", "banana"]: print(fruit).
Function
A reusable block of code defined with def that performs a specific task. Call it by name with parentheses: greet("Alice").
G
Generator
A function that uses yield instead of return to produce a sequence of values lazily (one at a time). Example: def count(): yield 1; yield 2.
global
A keyword that lets a function modify a variable defined at the module level. Example: global counter inside a function allows it to change the outer counter variable.
Global variable
A variable defined at the top level of a script or module, accessible from anywhere in that file. Use sparingly -- prefer passing values as function arguments.
H
Hash
A fixed-size numeric value computed from an object, used for fast lookups in dictionaries and sets. Only immutable types like str, int, and tuple are hashable.
help()
A built-in function that displays documentation for any object. Type help(print) in the interpreter to see how print() works.
I
IDE
Integrated Development Environment -- a software application for writing code, with features like syntax highlighting, debugging, and autocomplete. Popular Python IDEs include VS Code and PyCharm.
if
The keyword that starts a conditional statement. The indented block runs only when the condition is true. Example: if x > 0: print("positive").
Immutable
An object whose value cannot be changed after creation. Strings, integers, floats, and tuples are immutable in Python. Example: you cannot do "hello"[0] = "H".
Import
The import statement loads a module so you can use its functions and classes. Example: import math then math.sqrt(16).
Index
The position of an element in a sequence, starting from 0. In ["a", "b", "c"], the index of "b" is 1. Negative indices count from the end: -1 is the last item.
Inheritance
An OOP mechanism where a class (child) inherits attributes and methods from another class (parent). Example: class Cat(Animal): means Cat inherits from Animal.
input()
A built-in function that reads a line of text from the user. Example: name = input("Enter your name: "). Always returns a string.
Instance
A specific object created from a class. If Dog is a class, then my_dog = Dog("Rex") creates an instance of Dog.
Integer
A whole number without a decimal point, such as 42 or -7. Python integers have unlimited precision.
Interpreter
The program that reads and executes Python code line by line. You can run it interactively by typing python in a terminal, or use it to run scripts like python script.py.
Iterable
Any object you can loop over with a for loop. Lists, tuples, strings, dictionaries, sets, and generators are all iterables.
Iterator
An object that produces the next value in a sequence when you call next() on it. Created from an iterable with iter(). Example: it = iter([1, 2]); next(it) gives 1.
J
JSON
JavaScript Object Notation -- a lightweight text format for storing and exchanging data. Python's json module handles it: json.loads('{"key": "value"}') parses JSON into a dictionary.
join()
A string method that combines a list of strings into one, using the string as a separator. Example: ", ".join(["a", "b", "c"]) gives "a, b, c".
K
Key
The identifier used to access a value in a dictionary. In {"name": "Alice"}, "name" is the key. Keys must be immutable (strings, numbers, tuples).
Keyword
A reserved word in Python that has special meaning and cannot be used as a variable name. Examples: if, for, def, class, return, import.
Keyword argument
An argument passed to a function by name. Example: print("hi", end="") -- here end="" is a keyword argument.
L
Lambda
An anonymous (unnamed) function defined in a single expression using the lambda keyword. Example: square = lambda x: x ** 2 then square(4) gives 16.
len()
A built-in function that returns the number of items in an object. Example: len([1, 2, 3]) gives 3; len("hello") gives 5.
Library
A collection of modules and packages that provide pre-written functionality. Examples: the Python standard library, NumPy, Pandas, and Requests.
List
An ordered, mutable collection of items created with square brackets. Example: fruits = ["apple", "banana", "cherry"]. Access items by index: fruits[0].
List comprehension
A compact syntax for creating lists. Example: [x ** 2 for x in range(5)] gives [0, 1, 4, 9, 16]. Can include conditions: [x for x in range(10) if x % 2 == 0].
Local variable
A variable defined inside a function, accessible only within that function. It is created when the function runs and destroyed when it finishes.
Loop
A control structure that repeats a block of code. Python has two types: for loops (iterate over sequences) and while loops (repeat while a condition is true).
M
map()
A built-in that applies a function to every item in an iterable. Example: list(map(str, [1, 2, 3])) gives ["1", "2", "3"].
Method
A function that belongs to an object and is called using dot notation. Example: "hello".upper() calls the upper method on the string "hello", returning "HELLO".
Module
A single Python file (.py) containing functions, classes, and variables that can be imported. Example: import math imports the math module.
Mutable
An object whose value can be changed after creation. Lists, dictionaries, and sets are mutable. Example: my_list[0] = 99 changes the first element in place.
N
None
Python's null value, representing the absence of a value. Functions that do not explicitly return anything return None. Test for it with if x is None:.
NumPy
A popular third-party library for numerical computing with powerful array operations. Installed with pip install numpy. Example: import numpy as np; np.array([1, 2, 3]).
O
Object
Everything in Python is an object -- a piece of data with a type, value, and associated methods. Integers, strings, lists, functions, and classes are all objects.
open()
A built-in function to open a file and return a file object. Example: with open("data.txt", "r") as f: content = f.read().
Operator
A symbol that performs an operation on values. Arithmetic: +, -, *, /. Comparison: ==, !=, <, >. Logical: and, or, not.
P
Package
A directory of Python modules organized with an __init__.py file. Packages let you structure large projects. Example: import mypackage.mymodule.
Pandas
A powerful third-party library for data analysis and manipulation, built on top of NumPy. Install with pip install pandas. Its key data structure is the DataFrame.
Parameter
A variable listed in a function definition that receives a value when the function is called. In def greet(name):, name is the parameter. Compare with argument (the actual value passed).
pass
A statement that does nothing. Used as a placeholder when a block of code is required syntactically but you have nothing to write yet. Example: def todo(): pass.
PEP 8
The official Python style guide. It recommends conventions like 4-space indentation, snake_case for variables and functions, and PascalCase for class names.
pip
Python's package installer. Use it to install third-party libraries from PyPI. Example: pip install requests installs the Requests library.
print()
A built-in function that displays output to the console. Example: print("Hello, World!"). Accepts multiple arguments: print("a", "b", sep="-") outputs a-b.
Property
A way to define getter and setter methods on a class attribute using the @property decorator. Lets you access a method as if it were a simple attribute.
Python
A high-level, interpreted, general-purpose programming language created by Guido van Rossum. Known for its readable syntax, large standard library, and broad community support.
R
range()
A built-in that generates a sequence of numbers. Example: range(5) produces 0, 1, 2, 3, 4. Often used in for loops: for i in range(3):.
Recursion
When a function calls itself to solve a problem by breaking it into smaller sub-problems. Example: def factorial(n): return 1 if n == 0 else n * factorial(n - 1).
REPL
Read-Eval-Print Loop -- the interactive Python shell where you type expressions and see results immediately. Start it by running python in a terminal.
return
A statement that exits a function and sends a value back to the caller. Example: def add(a, b): return a + b. A function without return implicitly returns None.
S
Scope
The region of code where a variable is accessible. Python uses the LEGB rule: Local, Enclosing, Global, Built-in. A variable defined inside a function has local scope.
self
The conventional name for the first parameter of an instance method in a class. It refers to the current object. Example: def greet(self): print(self.name).
Set
An unordered collection of unique elements, created with curly braces or set(). Example: {1, 2, 3}. Useful for removing duplicates and set operations like union and intersection.
Slice
A way to extract a portion of a sequence using [start:stop:step] syntax. Example: "Python"[0:3] gives "Pyt"; [1, 2, 3, 4][::2] gives [1, 3].
sorted()
A built-in that returns a new sorted list from any iterable. Example: sorted([3, 1, 2]) gives [1, 2, 3]. Use reverse=True for descending order.
Standard library
The collection of modules included with every Python installation. Covers file I/O, math, JSON, CSV, regular expressions, networking, and much more -- no pip install needed.
String
A sequence of characters enclosed in quotes. Example: "Hello" or 'Hello'. Strings are immutable. Use triple quotes for multi-line strings: """line 1\nline 2""".
super()
A built-in function used inside a child class to call a method from the parent class. Example: super().__init__(name) calls the parent's constructor.
T
Traceback
The error report Python displays when an exception occurs. It shows the chain of function calls that led to the error, with the most recent call last. Read from the bottom up to find the cause.
True
One of the two Boolean values. Equivalent to 1 in numeric contexts. Non-empty strings, non-zero numbers, and non-empty collections are all "truthy."
try/except
A statement for handling exceptions. Code in the try block is attempted; if an error occurs, the except block runs instead. Example: try: int("abc") except ValueError: print("Not a number").
Tuple
An ordered, immutable sequence created with parentheses. Example: point = (3, 4). Like a list you cannot modify. Access items by index: point[0].
Type
The category of a value. Check with type(): type(42) returns <class 'int'>. Common types: int, float, str, list, dict, bool.
Type conversion
Changing a value from one type to another. Example: int("5") converts the string "5" to the integer 5. Also called type casting. Common conversions: str(), int(), float(), list().
V
Variable
A named reference to a value stored in memory. Created by assignment: age = 25. Variable names should be descriptive and follow snake_case convention.
venv
A built-in module for creating virtual environments. Usage: python -m venv myenv creates an isolated environment in the myenv folder.
Virtual environment
An isolated Python installation that has its own set of packages, separate from the system Python. Prevents version conflicts between projects. Created with python -m venv myenv and activated before use.
W
while loop
A loop that repeats a block of code as long as a condition remains true. Example: while count < 10: count += 1. Be careful to avoid infinite loops.
with statement
A context manager that ensures resources are properly set up and cleaned up. Most commonly used with files: with open("file.txt") as f: automatically closes the file when the block ends.
Z
Zero-based indexing
Python's convention where the first element of a sequence is at index 0, the second at 1, and so on. Example: in ["a", "b", "c"], "a" is at index 0.
zip()
A built-in that combines multiple iterables element by element into tuples. Example: list(zip([1, 2], ["a", "b"])) gives [(1, "a"), (2, "b")]. Useful for iterating over two lists in parallel.
