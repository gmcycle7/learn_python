MATLAB to Python — Quick Translation Table

A dense, scannable reference for converting MATLAB code to Python. Keep this page open while you code. All Python examples assume import numpy as np and import matplotlib.pyplot as plt where relevant.

How to Use This Page

Find what you want to do in the first column, then read across for the MATLAB and Python equivalents. Code is kept to 1–2 lines per cell for quick scanning. For detailed explanations and more examples, see MATLAB to Python.

std() differs by default!

MATLAB's std(v) uses N−1 (sample standard deviation). NumPy's np.std(v) uses N (population standard deviation) by default. To match MATLAB, use np.std(v, ddof=1).

The biggest operator swap

MATLAB: * = matrix multiply, .* = element-wise.
NumPy: @ = matrix multiply, * = element-wise.
This is the most common source of silent bugs when translating MATLAB to Python.

Matplotlib was designed to mimic MATLAB

The pyplot interface was intentionally modeled after MATLAB's plotting commands. Most function names are identical or nearly so. The biggest differences: prefix with plt., use keyword arguments instead of name-value pairs, and always call plt.show().

Import Cheat Sheet

Most MATLAB features are built in. In Python, you import libraries. Here are the standard imports for MATLAB-like work: 

# Standard scientific Python imports
import numpy as np              # arrays, math (like MATLAB core)
import matplotlib.pyplot as plt # plotting (like MATLAB figure/plot)
import pandas as pd             # tables/DataFrames (like MATLAB tables)
from scipy import linalg        # advanced linear algebra
from scipy import optimize      # optimization (like fmincon)
from scipy import integrate     # integration (like integral)
from scipy import signal        # signal processing (like DSP toolbox)
from scipy import stats         # statistics (like Statistics toolbox)
import os                       # file system operations
import math                     # basic math (sqrt, pi, etc.)
Tip: Start every scientific script with these

Put import numpy as np and import matplotlib.pyplot as plt at the top of every script. These two imports give you 90% of MATLAB's core functionality. Add import pandas as pd if you work with tables/CSV data.
